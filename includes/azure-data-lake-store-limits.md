Azure Data Lake Storage Gen1 è un repository su vasta scala a livello aziendale per carichi di lavoro di analisi di Big Data. È possibile utilizzare Data Lake Storage Gen1 per acquisire dati di qualsiasi dimensione, tipo e velocità di inserimento in un'unica posizione per l'analisi operativa ed esplorativa. Non esiste alcun limite alla quantità di dati che è possibile archiviare in un account Data Lake Storage Gen1.

| **Risorsa** | **Limite** | **Commenti** |
| --- | --- | --- |
| Numero massimo di account Gen1 di Archiviazione dati, per sottoscrizione, per area |10 | Per richiedere un aumento per questo limite, contattare il supporto tecnico. |
| Numero massimo di ACL di accesso, per file o cartella |32 | Si tratta di un limite rigido. Utilizzare i gruppi per gestire l'accesso con un numero inferiore di voci. |
| Numero massimo di ACL predefiniti, per file o cartella |32 | Si tratta di un limite rigido. Utilizzare i gruppi per gestire l'accesso con un numero inferiore di voci. |

### <a name="azure-active-directory-authentication"></a>Autenticazione di Azure Active Directory

Gli ancoraggi nello spazio di Azure consentono inoltre alle applicazioni di eseguire l'autenticazione con i token utente di Azure AD (Active Directory). Ad esempio, è possibile usare i token di Azure AD per l'integrazione con gli ancoraggi nello spazio di Azure. Se un'organizzazione gestisce gli utenti in Azure AD, è possibile fornire un token utente di Azure AD in Azure Spatial Anchors SDK. In questo modo è possibile eseguire l'autenticazione con il servizio degli ancoraggi nello spazio di Azure per un account che fa parte dello stesso tenant Azure AD.

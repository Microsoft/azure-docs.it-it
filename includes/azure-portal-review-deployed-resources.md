Verificare che sia possibile visualizzare i dati relativi alla macchina virtuale dall'interno del dashboard portale di Azure.

1. Nel portale di Azure selezionare **Dashboard**.

    :::image type="content" source="media/azure-portal-review-deployed-resources/navigate-to-dashboards.png" alt-text="passaggio al dashboard nel portale di Azure":::

1. Nella pagina del dashboard selezionare **Simple VM Dashboard**.

    :::image type="content" source="media/azure-portal-review-deployed-resources/select-simple-vm-dashboard.png" alt-text="Passare a Simple VM Dashboard":::

1. Esaminare il dashboard creato dal modello di Azure Resource Manager (modello ARM). Come si può notare, parte del contenuto è statico, ma sono presenti anche alcuni grafici che mostrano le prestazioni della VM creata all'inizio.

    :::image type="content" source="media/azure-portal-review-deployed-resources/review-simple-vm-dashboard.png" alt-text="Esaminare il dashboard Simple VM Dashboard":::

> [!NOTE]
> Lo <a a href="https://aka.ms/MRFeatureToolDocs" target="_blank">strumento di funzionalità della realtà mista</a> supporta attualmente solo Windows.

1. Seguire la documentazione <a href="https://aka.ms/MRFeatureToolDocs" target="_blank">dello strumento per la funzionalità di realtà mista</a> per configurare lo strumento e informazioni su come usarlo.
2. Installare il pacchetto della funzionalità "Microsoft Azure Object Anchors" nella cartella del progetto Unity.

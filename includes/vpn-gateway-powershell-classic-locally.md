Quando si usa il modello di distribuzione classica, non è possibile usare Azure Cloud Shell.When working with the classic deployment model, you can't use Azure Cloud Shell. È invece necessario installare la versione più recente dei cmdlet di PowerShell per La gestione dei servizi di Azure in locale nel computer. Questi cmdlet sono diversi dai cmdlet AzureRM o Az. Per installare i cmdlet SM, vedere [Installare i cmdlet di Service Management](/powershell/azure/servicemanagement/install-azure-ps). Per altre informazioni su Azure PowerShell in generale, vedere la documentazione di Azure PowerShell.For more information about Azure PowerShell in general, see the [Azure PowerShell documentation.](/powershell/azure/)

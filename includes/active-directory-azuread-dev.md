> [!IMPORTANT]
> [Microsoft Identity Platform (v2.0)](/azure/active-directory/develop/v2-overview) è un'evoluzione della piattaforma per sviluppatori di Azure Active Directory (Azure AD) (v1.0). Consente agli sviluppatori di creare applicazioni che supportano l'accesso per tutte le identità Microsoft e il recupero di token per chiamare API Microsoft, come Microsoft Graph o API create dagli sviluppatori.
> Questo contenuto è relativo all'endpoint Azure AD v1.0. Per i nuovi progetti è consigliabile usare l'endpoint v2.0. Per altre info, leggere [Perché eseguire l'aggiornamento a Microsoft Identity Platform (v2.0)?](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) nonché [Limitazioni di Microsoft Identity Platform](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations).

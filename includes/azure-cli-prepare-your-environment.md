## <a name="prepare-your-environment"></a>Preparare l'ambiente

1. Per eseguire i comandi dell'interfaccia della riga di comando di Azure, eseguire un'[installazione locale](/cli/azure/install-azure-cli) dell'interfaccia della riga di comando di Azure o avviare [Azure Cloud Shell](/azure/cloud-shell/quickstart) usando l'ambiente bash.

1. Se si usa un'installazione locale, accedere all'interfaccia della riga di comando usando il comando [az login](/cli/azure/reference-index#az-login).  Seguire le istruzioni visualizzate nel terminale per completare il processo di autenticazione.  Per altre opzioni di accesso, vedere [Accedere tramite l'interfaccia della riga di comando di Azure](/cli/azure/authenticate-azure-cli).

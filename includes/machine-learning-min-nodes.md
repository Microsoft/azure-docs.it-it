> [!IMPORTANT]
> Per evitare addebiti quando non è in esecuzione alcun processo, **impostare il numero minimo di nodi su 0**. Questa impostazione consente Azure Machine Learning di deallocare i nodi quando non sono in uso. Qualsiasi valore maggiore di 0 manterrà il numero di nodi in esecuzione, anche se non sono in uso.

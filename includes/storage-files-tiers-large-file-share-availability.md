Le condivisioni file standard con capacità 100 TiB presentano alcune limitazioni.

- Attualmente sono supportati solo gli account di archiviazione con ridondanza locale (LRS) e di archiviazione con ridondanza di zona.
- Dopo aver abilitato le condivisioni file di grandi dimensioni, non è possibile convertire gli account di archiviazione in account di archiviazione con ridondanza geografica (GRS) o con ridondanza geografica.
- Dopo aver abilitato condivisioni file di grandi dimensioni, non è possibile disabilitarle.

Le condivisioni file standard con limite di capacità 100 TiB sono disponibili a livello globale in tutte le aree di Azure, ad eccezione di:Standard file shares with 100 TiB capacity limit are available globally in all Azure regions, except:

- Archiviazione con ridondanza locale: tutte le aree, ad eccezione di Sud Africa Nord, Sud Africa ovest, Germania centro-occidentale e Germania nord.
- Archiviazione ridondante della zona: supportata per tutte le aree in cui è supportato lo spazio di archiviazione ridondante della zona, ad eccezione di Giappone orientale, Nord Europa, Sud Africa settentrionale.
- Archiviazione ridondante georidondinte/Geozona: non supportata.

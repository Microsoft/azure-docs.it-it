| Risorsa | Limite |
| --- | --- |
| Macchine virtuali per servizio cloud <sup>1</sup> |50 |
| Endpoint di input per servizio cloud <sup>2</sup> |150 |

<sup>1</sup> Le macchine virtuali create con il modello di distribuzione classico invece di Azure Resource Manager vengono archiviate automaticamente in un servizio cloud. È possibile aggiungere più macchine virtuali al servizio cloud per il bilanciamento del carico e disponibilità. 

<sup>2</sup> Gli endpoint di input consentono le comunicazioni con una macchina virtuale dall'esterno del servizio cloud della macchina virtuale. Le macchine virtuali nello stesso servizio cloud o rete virtuale può comunicare automaticamente tra loro.

Se la cache non è stata aggiunta al dashboard, è possibile trovarla nel [portale di Azure](https://portal.azure.com) usando **Tutti i servizi**.

![Pannello per la selezione di Cache Redis di Azure](media/redis-cache-browse/redis-cache-browse.png)

Per visualizzare le cache, fare clic su **Tutti i servizi** e cercare **Cache Redis**. 

Selezionare la cache desiderata per visualizzare e configurare le impostazioni per tale cache.

![Pannello per la selezione di una cache di Cache Redis di Azure](media/redis-cache-browse/redis-caches.png)

È possibile visualizzare e configurare la cache dal pannello **Cache Redis**.

![Tutte le impostazioni di Cache Redis di Azure](media/redis-cache-browse/redis-cache-blade.png)

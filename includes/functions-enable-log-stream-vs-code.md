Per attivare i log di flusso per l'app per le funzioni in Azure:

1. Premere F1 per aprire il riquadro comandi e quindi cercare ed eseguire il comando **Funzioni di Azure: Avvia log di flusso**.

1. Selezionare l'app per le funzioni in Azure e quindi selezionare **SÃ¬** per abilitare la registrazione delle applicazioni per l'app per le funzioni.

1. Attivare le funzioni in Azure. Tenere presente che i dati del log vengono visualizzati nella finestra di output in Visual Studio Code.

1. Al termine, ricordarsi di eseguire il comando **Funzioni di Azure: Arresta log di flusso** per disabilitare la registrazione per l'app per le funzioni.

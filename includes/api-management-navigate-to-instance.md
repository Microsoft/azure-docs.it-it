## <a name="go-to-your-api-management-instance"></a>Passare all'istanza di Gestione API

1. Nel portale di Azure cercare e selezionare **Servizi Gestione API**.

   ![Selezionare Servizi Gestione API](media/api-management-navigate-to-instance/view-apim1.png)

1. Nella pagina **Servizi Gestione API** selezionare l'istanza di Gestione API.

   ![Selezionare l'istanza di Gestione API](media/api-management-navigate-to-instance/view-apim2.png)

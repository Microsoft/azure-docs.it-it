> [!TIP]
> Il contenuto in questo articolo si applica all'archivio tabelle originale di Azure. Attualmente è tuttavia disponibile un'offerta alternativa per l'archiviazione di tabelle: l'API Tabella di Azure Cosmos DB. Questa API offre tabelle con disponibilità e prestazioni più elevate, distribuzione globale e indici secondari automatici. È disponibile anche in modalità [serverless](../articles/cosmos-db/serverless.md) a consumo. Esistono alcune [differenze di funzionalità](../articles/cosmos-db/table-api-faq.md#table-api-vs-table-storage) tra l'API Tabella in Azure Cosmos DB e archiviazione tabelle di Azure. Per altre informazioni, vedere [API di tabella di Azure Cosmos DB](../articles/cosmos-db/table-introduction.md).

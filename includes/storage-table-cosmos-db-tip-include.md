> [!TIP]
> Il contenuto in questo articolo si applica all'archivio tabelle originale di Azure. Attualmente è tuttavia disponibile un'offerta Premium per l'archivio tabelle: l'API Tabella di Azure Cosmos DB. Questa API offre tabelle ottimizzate per la velocità effettiva, distribuzione globale e indici secondari automatici. Esistono alcune [differenze di funzionalità tra l'API Tabella in Azure Cosmos DB e l'archivio tabelle di Azure](../articles/cosmos-db/faq.md#where-is-table-api-not-identical-with-azure-table-storage-behavior). Per altre informazioni e per provare l'esperienza Premium, vedere [API Tabella di Azure Cosmos DB](https://aka.ms/premiumtables).

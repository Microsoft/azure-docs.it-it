- Per accedere al servizio Data Box Edge, è necessario che l'organizzazione disponga di una sottoscrizione di Contratto Enterprise (EA) o Cloud Solution Provider (CSP). Per altre informazioni, vedere [iscriversi a una sottoscrizione di Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/).
- Poiché questo servizio di gestione è ospitato in Azure, è protetto dalle funzionalità di sicurezza di Azure. Per ulteriori informazioni sulle funzionalità di sicurezza fornite da Azure, visitare il [Centro protezione Microsoft Azure](https://azure.microsoft.com/support/trust-center/security/).
- Per le operazioni di gestione di SDK, è possibile ottenere la chiave di crittografia per la risorsa nelle **proprietà del dispositivo**. È possibile visualizzare la chiave di crittografia solo se si dispone delle autorizzazioni per la risorsa API Graph.

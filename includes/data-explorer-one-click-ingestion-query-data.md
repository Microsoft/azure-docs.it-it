## <a name="query-data"></a>Eseguire query sui dati

1. Selezionare il pulsante **v** per aprire la query. Copiare nell'interfaccia utente Web per modificare la query.

1. Nel menu a destra selezionare **Quick queries** (Query rapide) e **Tools** (Strumenti): 

    * **Quick queries** (Query rapide) include i collegamenti all'interfaccia utente Web con query di esempio.
    * **Tools** (Strumenti) include un collegamento a **Drop commands** (Comandi drop) nell'interfaccia utente Web, che consentono di risolvere problemi eseguendo i comandi `.drop` pertinenti.

    > [!TIP]
    > L'uso dei comandi `.drop` potrebbe causare perdite di dati. Prestare attenzione.

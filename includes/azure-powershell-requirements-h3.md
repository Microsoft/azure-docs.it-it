### <a name="requirements"></a>Requisiti

* Se si sceglie di usare PowerShell in locale:
  * [Installare il modulo AZ PowerShell](/powershell/azure/install-az-ps).
  * Connettersi al proprio account di Azure usando il cmdlet [Connect-AzAccount](/powershell/module/az.accounts/connect-azaccount) .
* Se si sceglie di usare Cloud Shell:
  * Per ulteriori informazioni, vedere [Panoramica delle Azure cloud Shell](https://docs.microsoft.com/azure/cloud-shell/overview) .

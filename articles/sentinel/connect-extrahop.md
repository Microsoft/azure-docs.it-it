# <a name="connect-extrahop-revealx"></a>Connettere ExtraHop Reveal(x)

Il connettore dati ExtraHop Reveal(x) consente di connettere facilmente il sistema Reveal(x) con Azure Sentinel per visualizzare i dashboard, creare avvisi personalizzati e migliorare l'analisi. Questa integrazione offre la possibilità di ottenere informazioni dettagliate sulla rete dell'organizzazione e di migliorare le funzionalità operative di sicurezza.


## <a name="prerequisites"></a>Prerequisiti

- Autorizzazioni di lettura e scrittura per l'area di lavoro di Azure Sentinel.Read and write permissions to the Azure Sentinel workspace.

- Appliance ExtraHop Discover o Command con firmware versione 7.8 o successiva con un account utente con privilegi illimitati (amministratore).

## <a name="connect-azure-sentinel-to-extrahop-revealx"></a>Connettere Azure Sentinel a ExtraHop Reveal(x)Connect Azure Sentinel to ExtraHop Reveal(x)

1. Nel portale di Azure passare a**Connettori dati** **di Azure Sentinel** > e quindi selezionare il connettore **ExtraHop Reveal(x).**

2. Selezionare **Apri pagina connettore**.

3. Seguire le istruzioni nella pagina **ExtraHop Reveal(x).**

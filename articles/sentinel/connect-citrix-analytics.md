# <a name="connect-citrix-analytics-security"></a>Connettere Citrix Analytics (Security) 

L'integrazione di Citrix Analytics (sicurezza) con Azure Sentinel consente di esportare i dati analizzati per eventi rischiosi da Citrix Analytics (sicurezza) in Azure Sentinel. È possibile creare dashboard personalizzati, analizzare i dati da altre origini insieme a quelli di Citrix Analytics (sicurezza) e creare flussi di lavoro personalizzati usando app per la logica per monitorare e mitigare gli eventi di sicurezza. 



## <a name="prerequisites"></a>Prerequisiti

- Autorizzazioni di lettura e scrittura per l'area di lavoro di Azure Sentinel.

- Diritto a Citrix Security Analytics in Citrix cloud.


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>Connettere Sentinel di Azure a Citrix Analytics (sicurezza)

1. Nel portale di Azure passare a**connettori dati** **sentinella** > di Azure e quindi selezionare il connettore **Citrix Analytics (sicurezza)** .

2. Selezionare **Apri connettore pagina**.

3. Seguire le istruzioni nella pagina **Citrix Analytics (sicurezza)** .

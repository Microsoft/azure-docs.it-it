# <a name="connect-citrix-analytics-security"></a>Connettere Citrix Analytics (Security) 

L'integrazione di Citrix Analytics (Sicurezza) con Azure Sentinel consente di esportare i dati analizzati per gli eventi rischiosi da Citrix Analytics (Sicurezza) in Azure Sentinel. È possibile creare dashboard personalizzati, analizzare i dati da altre origini insieme a quello di Citrix Analytics (Sicurezza) e creare flussi di lavoro personalizzati utilizzando App per la logica per monitorare e ridurre gli eventi di sicurezza. 



## <a name="prerequisites"></a>Prerequisiti

- Autorizzazioni di lettura e scrittura per l'area di lavoro di Azure Sentinel.Read and write permissions to the Azure Sentinel workspace.

- Diritto a Citrix Security Analytics in Citrix Cloud.


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>Connettere Azure Sentinel a Citrix Analytics (Sicurezza)Connect Azure Sentinel to Citrix Analytics (Security)

1. Nel portale di Azure passare a**Connettori dati** **di Azure Sentinel** > e quindi selezionare il connettore **Citrix Analytics (Sicurezza).**

2. Selezionare **Apri pagina connettore**.

3. Seguire le istruzioni nella pagina **Citrix Analytics (Sicurezza).**

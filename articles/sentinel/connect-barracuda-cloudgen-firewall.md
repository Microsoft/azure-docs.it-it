# <a name="connect-barracuda-cloudgen-firewall"></a>Connettere Barracuda CloudGen Firewall

Il connettore Barracuda CloudGen Firewall (CGFW) consente di connettere facilmente i log di Barracuda CGFW con Azure Sentinel, per visualizzare i dashboard, creare avvisi personalizzati e migliorare l'analisi. Ciò offre maggiori informazioni sulla rete dell'organizzazione e migliora le funzionalità di gestione della sicurezza.




## <a name="prerequisites"></a>Prerequisiti

- Autorizzazioni di lettura e scrittura per l'area di lavoro di Azure Sentinel.Read and write permissions to the Azure Sentinel workspace.

- Barracuda CloudGen Firewall deve essere configurato per esportare i log tramite Syslog.

## <a name="connect-azure-sentinel-to-barracuda-cloudgen-firewall"></a>Connettere Azure Sentinel a Barracuda CloudGen Firewall

1. Nel portale di Azure passare a Connettori dati **di Azure Sentinel** > e quindi selezionare il connettore **Barracuda CloudGen Firewall.In** the Azure portal, navigate to Azure Sentinel**Data connectors** and then select the Barracuda CloudGen Firewall connector.

2. Selezionare **Apri pagina connettore**.

3. Seguire le istruzioni nella pagina **Barracuda CloudGen Firewall.**

# <a name="connect-trend-micro-deep-security"></a>Connetti tendenza Micro Deep Security

Trend Micro Deep Security Connector consente di connettere facilmente i log di sicurezza approfonditi con Azure Sentinel, visualizzare i dashboard, creare avvisi personalizzati e migliorare l'analisi. In questo modo è possibile ottenere informazioni più dettagliate sulle reti e sui sistemi dell'organizzazione e migliorare le funzionalità di sicurezza.


## <a name="prerequisites"></a>Prerequisiti

- Autorizzazioni di lettura e scrittura per l'area di lavoro di Azure Sentinel.

## <a name="connect-azure-sentinel-to-trend-micro-deep-security"></a>Connettere Sentinel di Azure a Trend Micro Deep Security

1. Nel portale di Azure passare a connettori dati **sentinella di Azure**  >   e quindi selezionare il connettore per la **sicurezza di tendenza Micro Deep** .

2. Selezionare **Apri connettore pagina**.

3. Seguire le istruzioni e i passaggi successivi.

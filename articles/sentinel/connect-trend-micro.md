# <a name="connect-trend-micro-deep-security"></a>Connetti Trend Micro Deep Security

Il connettore Trend Micro Deep Security consente di connettere facilmente i log di Deep Security con Azure Sentinel, per visualizzare dashboard, creare avvisi personalizzati e migliorare l'analisi. Ciò offre maggiori informazioni sulle reti e sui sistemi dell'organizzazione e migliora le funzionalità delle operazioni di sicurezza.


## <a name="prerequisites"></a>Prerequisiti

- Autorizzazioni di lettura e scrittura per l'area di lavoro di Azure Sentinel.Read and write permissions to the Azure Sentinel workspace.

## <a name="connect-azure-sentinel-to-trend-micro-deep-security"></a>Connettere Azure Sentinel a Trend Micro Deep Security

1. Nel portale di Azure passare a Connettori dati **di Azure Sentinel** > e quindi selezionare il connettore Trend Micro Deep Security.In the Azure portal, navigate to Azure Sentinel**Data connectors** and then select the **Trend Micro Deep Security** connector.

2. Selezionare **Apri pagina connettore**.

3. Seguire le istruzioni e i passaggi successivi.

# <a name="connect-one-identity-safeguard"></a>Collegare una protezione dell'identità
Il connettore dati One Identity Safeguard migliora il connettore CEF (Common Event Format) standard con dashboard specifici per le sessioni con privilegi. Utilizzare questo connettore per iniziare facilmente a utilizzare gli eventi generati dal dispositivo per la visualizzazione, avvisi, indagini e altro ancora.


## <a name="prerequisites"></a>Prerequisiti

- Autorizzazioni di lettura e scrittura per l'area di lavoro di Azure Sentinel.Read and write permissions to the Azure Sentinel workspace.

## <a name="connect-azure-sentinel-to-one-identity-safeguard"></a>Connettere Azure Sentinel a una protezione dell'identitàConnect Azure Sentinel to One Identity Safeguard

1. Nel portale di Azure passare a Connettori dati **di Azure Sentinel** > e quindi selezionare il connettore One Identity Safeguard.In the Azure portal, navigate to Azure Sentinel**Data connectors** and then select the **One Identity Safeguard** connector.

2. Selezionare **Apri pagina connettore**.

3. Seguire le istruzioni nella pagina **One Identity Safeguard.**

## <a name="initialize-the-application"></a>Inizializzare l'applicazione

1. Creare un file JavaScript nell'ambiente di sviluppo integrato o nell'editor preferito e impostare i requisiti seguenti:

    ```javascript

    var FormData = require('form-data');
    var fs = require('fs');
    ```

2. Creare variabili per l'endpoint dell'API, la chiave di sottoscrizione e il percorso dell'immagine. `baseUri` puÃ² essere l'endpoint globale seguente o l'endpoint di [sottodominio personalizzato](../../../cognitive-services/cognitive-services-custom-subdomains.md) visualizzato nel portale di Azure per la risorsa:

    ```javascript

    var subscriptionKey = 'your-api-key';
    var imagePath = "path-to-your-image";
    ```

3. Creare una funzione denominata `requestCallback()` per stampare la risposta dall'API:


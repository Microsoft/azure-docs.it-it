#### <a name="c-nuget-package"></a>Pacchetto NuGet C++

È possibile installare C++ Speech SDK da **Gestione pacchetti** con il `Install-Package` comando seguente.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```

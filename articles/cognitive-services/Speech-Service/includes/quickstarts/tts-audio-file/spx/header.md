In questo argomento di avvio rapido si usa l'interfaccia della riga di comando per Voce per convertire la sintesi vocale archiviata in un file audio. Il servizio di sintesi vocale offre molte opzioni per le voci sintetizzate, come indicato nella sezione sulle [lingue supportate per la sintesi vocale](../../../../language-support.md#text-to-speech). Dopo una configurazione singola, l'interfaccia della riga di comando per Voce consente di sintetizzare la voce dal testo usando i comandi dalla riga di comando.

In questo argomento di avvio rapido si usa l'interfaccia della riga di comando per Voce dalla riga di comando per il riconoscimento vocale registrato in un file audio e produrre una trascrizione del testo. È facile usare l'interfaccia della riga di comando per Voce per eseguire attività comuni di riconoscimento, ad esempio la trascrizione di conversazioni. Dopo una configurazione singola, l'interfaccia della riga di comando per Voce consente di trascrivere l'audio in testo in modo interattivo con un microfono o da file usando uno script batch.

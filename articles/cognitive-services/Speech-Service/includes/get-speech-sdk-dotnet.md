#### <a name="c-nuget-package"></a>Pacchetto NuGet di C

.NET Speech SDK può essere installato dall'interfaccia `dotnet add` della riga di comando **di .NET Core** con il comando seguente.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

.NET Speech SDK può essere installato da `Install-Package` Gestione **pacchetti** con il comando seguente.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```

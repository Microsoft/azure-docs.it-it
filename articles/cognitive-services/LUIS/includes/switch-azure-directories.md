1. Fare clic sull'avatar nell'angolo superiore destro della schermata. Quindi fare clic su **passa a un'altra directory di Azure**. 
2. Nella finestra visualizzata assicurarsi di selezionare la directory di Azure che contiene la risorsa LUIS che Ã¨ stata condivisa con l'utente.

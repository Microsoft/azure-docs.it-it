# <a name="authoring-api-changes"></a>Modifica dell'API di creazione 

Le API di creazione vengono aggiornate, per V3, per consentire la scomposizione del [modello con entità apprese automaticamente.](luis-concept-data-extraction.md) 

È possibile creare nell'API V2 o anteprima V3. La stessa app può anche scegliere di usare le risposte dell'endpoint di stima V2 o V3.The same app can also choose to use V2 or V3 prediction endpoint responses. Non è necessario disporre della stessa versione dell'API per la creazione e la risposta dell'endpoint di stima.

Il portale LUIS utilizza esclusivamente le API di creazione V3.

## <a name="next-steps"></a>Passaggi successivi

Esaminare [le modifiche all'endpoint](luis-migration-api-v3.md) di stima per l'API V3.

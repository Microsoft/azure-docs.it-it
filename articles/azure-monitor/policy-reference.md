# <a name="azure-policy-built-in-definitions-for-azure-monitor"></a>Definizioni predefinite di Criteri di Azure per Monitoraggio di Azure

Questa pagina include un indice delle definizioni di criteri predefiniti di [Criteri di Azure](../governance/policy/overview.md) per Monitoraggio di Azure.

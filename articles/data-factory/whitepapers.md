# <a name="azure-data-factory-whitepapers"></a>White paper di Azure Data Factory 

I white paper consentono di esplorare Azure Data Factory a un livello più profondo. Questo articolo fornisce un elenco di white paper disponibili per Azure Data Factory.This article provides you with a list of available whitepaper for Azure Data Factory.

| **Whitepaper** | **Descrizione** |

|[Azure Data Factory: integrazione dei dati nel cloud](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | Questo documento descrive in che modo Azure Data Factory può consentire la creazione di un data warehouse moderno, l'abilitazione dell'analisi avanzata per guidare applicazioni SaaS intelligenti e portare i pacchetti di SQL Server Integrations Services in Azure.This paper describes how Azure Data Factory can enable you to build a modern data warehouse, enable advanced analytics to drive intelligent SaaS applications and lift your SQL Server Integrations Services packages to Azure.|

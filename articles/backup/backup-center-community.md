# <a name="access-community-resources-using-backup-center"></a>Accedere alle risorse della community con backup Center

È possibile usare il centro di backup per accedere a diverse risorse della community utili per un amministratore o un operatore di backup.

## <a name="using-community-hub"></a>Utilizzo di hub della community

Per accedere al hub della community, passare a backup Center nel portale di Azure e selezionare la voce di menu **community** .

![Hub della community](./media/backup-center-community/backup-center-community-hub.png)

Di seguito sono riportate alcune delle risorse disponibili tramite la hub della community:

- **Microsoft Q&a**: è possibile usare questo forum per porre domande su diverse funzionalità del prodotto e ottenere informazioni aggiuntive sulla community.

- **Richieste di funzionalità**: è possibile passare a UserVoice e alle richieste di funzionalità di file.

- **Esempi per le distribuzioni automatiche**: usando il hub della community, è possibile individuare modelli di Azure Resource Manager di esempio (ARM) e criteri di Azure che è possibile usare. È anche possibile trovare script di PowerShell di esempio, comandi dell'interfaccia della riga di comando e script di backup di database Microsoft.

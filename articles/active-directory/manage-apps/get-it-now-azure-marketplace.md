# <a name="get-it-now---add-an-app-from-the-azure-marketplace"></a>Scarica ora - Aggiungere un'app da Azure Marketplace

L'operazione è quasi completata. 

Se si sta provando a usare Azure AD come provider di identità per un'app, questa è la pagina corretta. È sufficiente aggiungerlo al tenant di Azure AD. Per informazioni su come eseguire questa operazione, seguire la serie di guide di avvio rapido disponibili qui: [Visualizzare le app nel tenant di Azure AD](view-applications-portal.md).

Se si sta tentando di installare un'app nel computer locale o nel dispositivo mobile, questa è la pagina sbagliata. Azure Marketplace è progettato per le organizzazioni che usano app con i tenant di Azure. Per i computer e i dispositivi personali, consultare [Microsoft App Store](https://www.microsoft.com/store/apps).

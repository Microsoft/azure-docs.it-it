# <a name="access-azure-kinect-dk-microphone-input-data"></a>Accedere ai dati di input del microfono Kinect DK di Azure

Le [guide introduttive per l'SDK di riconoscimento vocale](https://docs.microsoft.com/azure/cognitive-services/speech-service) forniscono esempi di come usare la matrice di microfoni Kinect dk di Azure in diversi linguaggi di programmazione.
Ad esempio, vedere **riconoscimento vocale in C++ in Windows tramite la Guida introduttiva all'SDK di riconoscimento vocale** . Il codice è disponibile [in GitHub](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/cpp).

Accedere alla matrice di microfoni anche tramite l'API Windows. Per informazioni dettagliate sulla documentazione di Windows, vedere i documenti seguenti:

* [Architettura audio Windows](https://docs.microsoft.com/windows-hardware/drivers/audio/windows-audio-architecture)
* [Documentazione di Windows. Media. Capture](https://docs.microsoft.com/uwp/api/Windows.Media.Capture)
* [Esercitazione per l'acquisizione di webcam](https://docs.microsoft.com/windows/uwp/audio-video-camera/basic-photo-video-and-audio-capture-with-mediacapture)
* [Informazioni audio USB](https://docs.microsoft.com/windows-hardware/drivers/audio/usb-2-0-audio-drivers)

È anche possibile esaminare le [specifiche hardware della matrice di microfoni](hardware-specification.md#microphone-array).

# <a name="versioning-and-support-for-azure-defender-for-iot"></a>Controllo delle versioni e supporto per Azure Defender 

Questo articolo descrive il supporto, i criteri di modifica di rilievo per Defender for Internet e le versioni di Azure Defender per le cose attualmente disponibili. 

## <a name="servicing-information-and-timelines"></a>Informazioni e tempistiche di manutenzione 

Microsoft prevede di rilasciare gli aggiornamenti per Azure Defender per tutto il tempo non più di una volta al trimestre. Ogni versione disponibile a livello generale (GA) di Azure Defender per il sensore Internet e Azure Defender per la console di gestione locale è supportata per un massimo di nove mesi dopo il relativo rilascio. Le correzioni e le nuove funzionalità verranno applicate alla versione corrente di GA attualmente supportata e non verranno applicate alle versioni GA precedenti.

## <a name="versions-and-support-dates"></a>Versioni e date di supporto

| Versione | Data di rilascio | Data di fine supporto |
|--|--|--|
| 10,0 | 01/2021 | 10/2021 |

# <a name="quickstart-prepare-your-network-for-azure-defender-for-iot-deployment"></a>Avvio rapido: Preparare la rete per la distribuzione di Azure Defender per IoT

## <a name="choose-your-hardware"></a>Scegliere l'hardware
Azure Defender per IoT supporta varie appliance certificate, tra cui:
- Appliance certificate preconfigurate, in cui è già installato il software
- Appliance certificate non configurate in cui è possibile scaricare e installare il software dei sensori
- Specifiche dell'appliance della console di gestione locale. La console di gestione locale non è disponibile come appliance preconfigurata. 

Per altre informazioni, vedere la [guida alle specifiche hardware](https://aka.ms/AzureDefenderforIoTBareMetalAppliance)


## <a name="install-components"></a>Installazione dei componenti
Dopo aver acquisito i componenti hardware e scaricato il pacchetto software dal portale di Azure Defender per IoT, è possibile installare il software dei componenti. Per altre informazioni, vedere la [Guida all'installazione](https://aka.ms/AzureDefenderforIoTInstallSensorISO).

## <a name="onboard-sensors-to-the-azure-defender-for-iot-portal"></a>Eseguire l'onboarding dei sensori nel portale di Azure Defender per IoT 
 Questo articolo fornisce una panoramica dei processi di distribuzione, onboarding e aggiornamento per i sensori di Azure Defender per IoT e la console di gestione locale. Questo processo viene completato seguendo le istruzioni riportate nella pagina iniziale di Azure Defender per IoT e in quelle successive.

![Screenshot della pagina iniziale di Azure](media/updates/image4.png)

Per aprire la pagina iniziale:

1. Accedere al proprio account Azure e cercare Azure Defender per IoT.

1. Per la **distribuzione dei sensori** , è necessario acquistare hardware preconfigurato da Arrow oppure acquistare un dispositivo in cui installare il software dei sensori.
1. Per la **distribuzione della console di gestione locale** è necessario acquistare un dispositivo e installarvi il software della console di gestione locale. Per altre informazioni.




## <a name="network-setup"></a>Configurazione della rete
Dopo aver installato i componenti di sistema di Defender per IoT, è possibile configurare le apparecchiature per il monitoraggio ottimale della rete. Per altre informazioni, vedere la [guida alla configurazione di rete](https://aka.ms/AzureDefenderForIoTNetworkSetup).

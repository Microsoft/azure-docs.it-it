# <a name="micro-focus-devtest-environments-on-azure"></a>Micro ambienti di sviluppo/test con lo stato attivo in Azure

Iniziare a usare Azure con un ambiente di sviluppo e test è un ottimo modo per migliorare le attività di modernizzazione. Le applicazioni mainframe che dipendono da sistemi legacy come IBM CICS (Customer Information Control System) e IMS (Information Management System) possono essere riallocate in Azure tramite soluzioni partner.

Micro Focus ha collaborato con Microsoft per offrire strumenti di migrazione e riallocazione del mainframe. È possibile eseguire applicazioni mainframe in scenari di sviluppo, test e produzione in Azure e nella piattaforma Microsoft Windows.

Micro Focus offre soluzioni di sviluppo e riallocazione del test che estendono le applicazioni aziendali COBOL e PL/I in Azure.

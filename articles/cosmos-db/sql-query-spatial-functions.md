# <a name="spatial-functions-azure-cosmos-db"></a>Funzioni spaziali (Azure Cosmos DB)
[!INCLUDE[appliesto-sql-api](includes/appliesto-sql-api.md)]

Cosmos DB supporta le seguenti funzioni predefinite di Open Geospatial Consortium (OGC) per l'esecuzione di query geospaziali. 

## <a name="functions"></a>Funzioni

Le seguenti funzioni scalari eseguono un'operazione su un valore di input di oggetto spaziale e restituiscono un valore numerico o booleano.  

* [ST_DISTANCE](sql-query-st-distance.md)
* [ST_INTERSECTS](sql-query-st-intersects.md)
* [ST_ISVALID](sql-query-st-isvalid.md)
* [ST_ISVALIDDETAILED](sql-query-st-isvaliddetailed.md)
* [ST_WITHIN](sql-query-st-within.md)

# <a name="toggle-live-chat"></a>Attivare e disattivare la live chat

Questo articolo illustra come attivare o disattivare la chat dal vivo nell'applicazione IoT Central. È possibile utilizzare la chat dal vivo per accedere al supporto tecnico.

> [!NOTE]
> L'opzione di chat è disponibile solo per le applicazioni create utilizzando il piano tariffario gratuito.

## <a name="chat-with-us"></a>Chatta con noi

Per ottenere supporto tecnico, apri l'applicazione IoT Central e seleziona **Chattate con noi**.

![Chatta con noi](media/howto-show-hide-chat/chat-with-us.png)

È possibile digitare una domanda come illustrato nella schermata seguente:You can type a question as shown in the following screenshot:

![Modulo "Configure Device Details" (Configura dettagli dispositivo) con i dettagli per impostazioni e proprietà](media/howto-show-hide-chat/sample-chat.png)

## <a name="hide-chat"></a>Nascondi chat

Per nascondere la chat, scegli **Nascondi chat** nel pannello **Aiuto:**

 ![Nascondi chat](media/howto-show-hide-chat/hide-chat.png)

## <a name="enable-chat"></a>Abilita chat

Per visualizzare la chat, scegli **Mostra chat** nel pannello **Aiuto:**

 ![Mostra chat](media/howto-show-hide-chat/show-chat.png)

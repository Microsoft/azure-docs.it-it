# <a name="getting-started-with-azure-iot-embedded-device-development"></a>Introduzione allo sviluppo Azure IoT dispositivi incorporati

**Si applica a:** [Sviluppo di dispositivi incorporati](about-iot-develop.md#embedded-device-development)

Questa guida introduttiva contiene un set di guide introduttive che illustra come iniziare a usare dispositivi incorporati e Azure IoT. 

In ogni guida introduttiva vengono completate le attività di base seguenti:
* Installare un set di strumenti di sviluppo incorporati per la programmazione di un dispositivo specifico in C
* Compilare un'immagine che include Azure RTOS componenti ed esempi e quindi eseguire il flash di un dispositivo
* Connettere in modo sicuro un dispositivo a Azure IoT
* Visualizzare i dati di telemetria dei dispositivi, visualizzare le proprietà e richiamare i metodi da cloud a dispositivo

# <a name="getting-started-with-azure-iot-embedded-device-development"></a>Introduzione allo sviluppo di dispositivi embedded per Azure

**Si applica a**: [sviluppo di dispositivi incorporati](about-iot-develop.md#embedded-device-development)

Questa Guida introduttiva contiene una serie di guide introduttive che illustrano come iniziare a usare dispositivi incorporati e Azure. 

In ogni avvio rapido vengono completate le attività di base seguenti:
* Installare un set di strumenti di sviluppo incorporati per la programmazione di un dispositivo specifico in C
* Creare un'immagine che includa componenti ed esempi di Azure RTO, quindi eseguire il flashing di un dispositivo
* Connetti in modo sicuro un dispositivo ad Azure
* Visualizzare i dati di telemetria dei dispositivi, visualizzare le proprietà e richiamare metodi da cloud a dispositivo

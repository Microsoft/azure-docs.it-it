# <a name="troubleshoot-azure-migrate"></a>Risolvere i problemi relativi ad Azure Migrate

[Azure Migrate](migrate-services-overview.md) offre un hub di strumenti per la valutazione e la migrazione, nonché offerte di fornitori di software indipendenti di terze parti (ISV). Questo articolo consente di risolvere i problemi relativi a Azure Migrate, Azure Migrate Server Assessment e Azure Migrate Server Migration.This article helps you troubleshoot issues with Azure Migrate, Azure Migrate Server Assessment, and Azure Migrate Server Migration.

## <a name="how-do-i-create-or-find-a-project"></a>Come faccio a creare o trovare un progetto?

Consultare la guida alla [risoluzione dei problemi del progetto Azure Migrate](troubleshoot-project.md).

## <a name="i-cant-get-the-appliance-working"></a>Non riesco a far funzionare l'apparecchio

Esaminare [le risposte ai problemi comuni](troubleshoot-appliance-discovery.md) relativi alla distribuzione dell'appliance.

## <a name="machines-arent-discovered"></a>Le macchine non vengono scoperte

Esaminare i problemi di individuazione comuni.

## <a name="app-discovery-isnt-working"></a>L'individuazione delle app non funziona

L'individuazione di app, ruoli e funzionalità in esecuzione nei computer locali è attualmente supportata solo per le macchine virtuali VMware.Discovery of apps, roles, and features running on-premises machines is currently only supported for VMware VMs. [Esaminare gli errori comuni](troubleshoot-appliance-discovery.md#common-app-discovery-errors) per l'individuazione delle app.

## <a name="assessment-isnt-working"></a>La valutazione non funziona

[Esaminare i](troubleshoot-assessment.md) problemi e gli errori di valutazione comuni.

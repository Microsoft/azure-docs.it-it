# <a name="troubleshoot-azure-migrate"></a>Risolvere i problemi relativi ad Azure Migrate

[Azure migrate](migrate-services-overview.md) offre un hub di strumenti per la valutazione e la migrazione, nonché offerte di fornitori di software indipendenti (ISV) di terze parti. Questo articolo consente di risolvere i problemi relativi alla Azure Migrate, alla valutazione Azure Migrate server e alla migrazione di Azure Migrate server.

## <a name="how-do-i-create-or-find-a-project"></a>Ricerca per categorie creare o trovare un progetto?

Esaminare la [Guida alla risoluzione dei problemi di Azure migrate Project](troubleshoot-project.md).

## <a name="i-cant-get-the-appliance-working"></a>Non è possibile far funzionare l'appliance

Esaminare le [risposte ai problemi comuni](troubleshoot-appliance-discovery.md) con la distribuzione dell'appliance.

## <a name="machines-arent-discovered"></a>Computer non individuati

Esaminare i [problemi di individuazione comuni](common-questions-discovery-assessment.md).

## <a name="app-discovery-isnt-working"></a>App-Discovery non funziona

L'individuazione di app, ruoli e funzionalità in esecuzione in computer locali è attualmente supportata solo per le macchine virtuali VMware. [Esaminare gli errori comuni](troubleshoot-appliance-discovery.md#common-app-discovery-errors) per l'individuazione delle app.

## <a name="assessment-isnt-working"></a>Valutazione non funzionante

[Esaminare](troubleshoot-assessment.md) i problemi di valutazione comuni e gli errori.

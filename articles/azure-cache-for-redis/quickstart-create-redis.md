# <a name="quickstart-create-an-open-source-redis-cache"></a>Guida introduttiva: creare una cache Redis Open Source

La cache di Azure per Redis offre una soluzione [Redis open source](https://redis.io/) completamente gestita in Azure. È possibile iniziare con un'istanza della cache di Azure per Redis di qualsiasi livello (Basic, Standard o Premium) e dimensioni, quindi ampliarla in base a specifiche esigenze di prestazioni delle applicazioni. Questo argomento di avvio rapido illustra come usare il portale di Azure per creare una nuova cache di Azure per Redis.

## <a name="prerequisites"></a>Prerequisiti

Prima di iniziare, è necessaria una sottoscrizione di Azure. Se non se ne ha una, creare un [account gratuito](https://azure.microsoft.com/free/).

# <a name="change-to-full-screen-view-for-a-vm-session-azure-bastion"></a>Passare alla visualizzazione a schermo intero per una sessione di macchina virtuale: Azure BastionChange to full screen view for a vm session: Azure Bastion

Questo articolo consente di modificare la visualizzazione della macchina virtuale a schermo intero e viceversa nel browser. Prima di usare una macchina virtuale, assicurarsi di aver seguito la procedura per [creare un host Bastion](bastion-create-host-portal.md). Connettersi quindi alla macchina virtuale che si vuole usare utilizzando [RDP](bastion-connect-vm-rdp.md) o [SSH](bastion-connect-vm-ssh.md).

## <a name="launch-the-clipboard-tool"></a>Avviare lo strumento Appunti

Durante la sessione remota, avviare la tavolozza degli strumenti di accesso agli Appunti del Bastion selezionando le due frecce che si trovano al centro sinistro della sessione.

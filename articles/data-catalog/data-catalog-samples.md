# <a name="azure-data-catalog-developer-samples"></a>Esempi per sviluppatori di Azure Data Catalog

Iniziare a sviluppare app di Azure Data Catalog usando l'API REST di Data Catalog.Get started developing Azure Data Catalog apps using the Data Catalog REST API. L'API REST Data Catalog è un'API basata su REST che consente l'accesso a livello di codice alle risorse di Data Catalog per registrare, annotare e cercare asset di dati.

## <a name="samples-available-on-githubcom"></a>Campioni disponibili su GitHub.com

* [Introduzione al Catalogo dati di Azure](https://github.com/Azure-Samples/data-catalog-dotnet-get-started/)
  
   The get started sample shows you how to authenticate with Azure AD to Register, Search, and Delete a data asset using the Data Catalog REST API.
   
* [Introduzione ad Azure Data Catalog usando l'entità servizioGet started with Azure Data Catalog using Service Principal](https://github.com/Azure-Samples/data-catalog-dotnet-service-principal-get-started/)

   Questo esempio illustra come registrare, cercare ed eliminare un asset di dati usando l'API REST di Data Catalog.This sample shows you how to register, search, and delete a data asset using the Data Catalog REST API. In questo esempio viene utilizzata l'autenticazione dell'entità servizio.

* [Strumento Di importazione/esportazione per Azure Data Catalog](https://github.com/Azure-Samples/data-catalog-dotnet-import-export/)

   Questo esempio che illustra come usare l'API REST di Data Catalog per recuperare gli asset da Azure Data Catalog e serializzarli in un file. Mostra anche come prelevare un set di asset serializzato come JSON ed eseguirne il push nel catalogo. Supporta l'esportazione di un subset del catalogo usando una query di ricerca.

* [Registrazione in blocco e annotazione in Azure Data Catalog](https://github.com/Azure-Samples/data-catalog-dotnet-excel-register-data-assets/)
  
   Questo esempio illustra come registrare in blocco gli asset di dati da una cartella di lavoro di Excel usando l'API REST di Data Catalog e Open XML.
  
* [Termini del glossario per l'importazione in blocco in Azure Data Catalog](https://github.com/Azure-Samples/data-catalog-bulk-import-glossary/)

   In questo esempio viene illustrato come importare i termini di glossario da un file CSV in un glossario ADC.

* [Bulk import relationships into Azure Data Catalog](https://github.com/Azure-Samples/data-catalog-bulk-import-relationship/)

   In questo esempio viene illustrato come importare a livello di codice le informazioni sulle relazioni da un file CSV in un catalogo dati.

* [Pubblicare relazioni in Azure Data CatalogPublish relationships into Azure Data Catalog](https://github.com/Azure-Samples/data-catalog-dotnet-publish-relationships/)

   In questo esempio viene illustrato come pubblicare a livello di codice le informazioni sulle relazioni in un catalogo dati.

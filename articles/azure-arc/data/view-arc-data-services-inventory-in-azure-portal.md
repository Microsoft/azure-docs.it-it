# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Visualizzare l'inventario delle istanze nel portale di Azure

Dopo aver caricato le [metriche, i log](upload-metrics-and-logs-to-azure-monitor.md)o l' [utilizzo](view-billing-data-in-azure.md), è possibile visualizzare l'istanza come risorsa di Azure dalla portale di Azure. A tale scopo, passare all'URL speciale [https://portal.azure.com](https://portal.azure.com) e seguire gli screenshot seguenti

1. Passa a tutti i servizi
1. Cercare il tipo di istanze di database tipi
1. Aggiungila ai Preferiti
1. Fare clic su tipo di istanza dal NAV di sinistra
1. Consente di visualizzare le istanze nella stessa visualizzazione delle altre risorse di Azure SQL o dell'iperscalabilità di Azure PostgreSQL (usare i filtri per ottenere una visualizzazione granulare)

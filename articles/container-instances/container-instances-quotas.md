# <a name="quotas-and-limits-for-azure-container-instances"></a>Quote e limiti per istanze di Azure Container

Tutti i servizi di Azure prevedono determinati limiti predefiniti e quote per le risorse e le funzionalità. Questo articolo descrive le quote e i limiti predefiniti per le istanze di Azure Container.

La disponibilità di risorse di calcolo, memoria e archiviazione per le istanze del contenitore di Azure varia in base all'area e al sistema operativo. Per informazioni dettagliate, vedere [Disponibilità delle risorse per le istanze del contenitore](container-instances-region-availability.md)di Azure.For details, see Resource availability for Azure Container Instances.

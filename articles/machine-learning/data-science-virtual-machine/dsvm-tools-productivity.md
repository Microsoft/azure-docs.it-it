# <a name="productivity-tools-on-the-data-science-virtual-machine"></a>Strumenti di produttività nel Data Science Virtual Machine

Oltre ai data science e agli strumenti di programmazione, DSVM contiene strumenti di produttività che consentono di acquisire e condividere informazioni dettagliate con i colleghi. Microsoft 365 app per Enterprise è l'esperienza di Office più produttiva e sicura per le aziende, consentendo ai team di collaborare senza problemi ovunque e in qualsiasi momento. Con Power BI Desktop è possibile passare dai dati alle informazioni dettagliate per l'azione. Il browser Microsoft Edge è un Web browser moderno, veloce e sicuro.

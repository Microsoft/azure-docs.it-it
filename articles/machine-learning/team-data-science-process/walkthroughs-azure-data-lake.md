# <a name="azure-data-lake-data-science-walkthroughs-using-u-sql"></a>Procedure dettagliate di data science per Azure Data Lake con U-SQL

Queste procedure dettagliate usano U-SQL con Azure Data Lake per eseguire analisi predittive. Seguono i passaggi descritti nel processo di data science per i team. Per una panoramica del processo di data science per i team, vedere [Processo di data science](overview.md). Per un'introduzione ad Azure Data Lake, vedere [Panoramica di Archivio Azure Data Lake](../../data-lake-store/data-lake-store-overview.md).

Procedure dettagliate di data science aggiuntive che eseguono il processo di analisi scientifica dei dati del team sono raggruppate in base alla **piattaforma** in uso. Per i dettagli di questi esempi, vedere [Procedure dettagliate del processo di data science per i team](walkthroughs.md).


## <a name="predict-taxi-tips-using-u-sql-with-azure-data-lake"></a>Stimare le mance dei taxi con U-SQL e Azure Data Lake

Nella procedura dettagliata [Usare Azure Data Lake per l'analisi scientifica dei dati](data-lake-walkthrough.md) viene illustrato come usare Azure Data Lake per eseguire attività di esplorazione dei dati e classificazione binaria.  I dati sono un esempio del set di dati dei taxi di New York.  L'attività prevede se una mancia viene pagata o meno da un cliente.

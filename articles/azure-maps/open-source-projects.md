Queste iniziative open source guidate dalla community vengono create e gestite dal team di Azure Maps.This open-source, community-driven initiatives are created and maintained by the Azure Maps team. Non fanno parte dell'offerta di prodotti o servizi standard.

Di seguito sono elencati alcuni dei più popolari esempi e progetti open source di Azure Maps.

| [Azure Maps Gov Cloud Code Samples](https://github.com/Azure-Samples/AzureMapsCodeSamples) | Raccolta di esempi di codice per l'uso di Azure Maps tramite Azure Government Cloud.A collection of code samples for using Azure Maps through Azure Government Cloud. |

## <a name="supportability-of-open-source-projects"></a>Supportabilità di progetti open source

Le istruzioni seguenti si applicano a tutti gli esempi e progetti open source creati e gestiti da Azure Maps:The following statements apply across all of the Azure Maps created and maintained open-source projects and samples:

- I progetti e gli esempi open source di Azure Maps vengono creati da Microsoft e dalla community.
- I progetti e gli esempi open source di Azure Maps vengono gestiti da Microsoft e dalla community.
- I progetti e gli esempi open source di Azure Maps usano tecniche supportate e consigliate.
- I progetti e gli esempi open source di Azure Maps sono un'iniziativa della community, ovvero le persone che lavorano all'iniziativa a beneficio degli altri e hanno anche il normale lavoro quotidiano.
- I progetti e gli esempi open source di Azure Maps NON sono un prodotto e pertanto non sono supportati dal supporto Premier o da altri canali di supporto ufficiali.
- I progetti e gli esempi open source di Azure Maps sono supportati in modo simile a altri progetti open source eseguiti da Microsoft con il supporto della community da parte della community.

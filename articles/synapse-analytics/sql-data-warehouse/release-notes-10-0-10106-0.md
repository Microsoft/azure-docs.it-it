# <a name="azure-synapse-analytics-release-notes"></a>Note sulla versione di Azure Synapse AnalyticsAzure Synapse Analytics release notes

Questo articolo riepiloga le nuove funzionalità e i miglioramenti nelle versioni recenti di [SQL Analytics](sql-data-warehouse-overview-what-is.md) in Azure Synapse. L'articolo elenca anche importanti aggiornamenti del contenuto che non sono direttamente correlati alla versione, ma pubblicati nello stesso intervallo di tempo. Per miglioramenti ad altri servizi di Azure, vedere [Aggiornamenti dei servizi](https://azure.microsoft.com/updates).

## <a name="check-your-azure-synapse-version"></a>Controllare la versione di Azure SynapseCheck your Azure Synapse version

Man mano che le nuove funzionalità vengono distribuite in tutte le aree, controllare la versione distribuita nell'istanza e le note sulla versione più recenti per verificarne la disponibilità delle funzionalità. Per verificare la versione, connettersi al pool SQL tramite `SELECT @@VERSION;` SQL Server Management Studio (SSMS) ed eseguire per restituire la versione corrente.

Utilizzare la versione identificata per verificare quale versione è stata applicata al pool SQL. La data nell'output identifica il mese per il rilascio applicato al pool SQL.

> [!NOTE]
> Il nome del prodotto@VERSION restituito da SELECT : cambierà da Microsoft Azure SQL Data Warehouse ad Azure Synapse Analytics. Invieremo un avviso anticipato prima che venga apportata la modifica. Questa modifica è rilevante per i clienti che@VERSION analizzano il nome del prodotto dal risultato di SELECT nel codice dell'applicazione. Per evitare modifiche al codice dell'applicazione a causa del rebranding del prodotto, utilizzare questi comandi per eseguire una query SERVERPROPERTY per il nome e la versione del prodotto del database: per restituire il numero di versione XX. X.XXXXX. X (senza nome prodotto) utilizzare questo comando: 
> ```
> SELECT SERVERPROPERTY('ProductVersion')
> ```
> Per restituire l'edizione del motore, usare questo comando che restituisce 6 per Azure Synapse Analytics (in precedenza SQL Data Warehouse):To return engine edition, use this command that returns 6 for Azure Synapse Analytics (Formerly SQL Data Warehouse): 
> ```
> SELECT SERVERPROPERTY('EngineEdition')
> ```

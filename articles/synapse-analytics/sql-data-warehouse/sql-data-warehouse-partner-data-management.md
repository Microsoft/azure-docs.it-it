# <a name="azure-synapse-analytics-data-management-partners"></a>Partner di gestione dei dati di Azure Synapse Analytics

Per creare una soluzione di data warehouse con la funzionalità di pool Synapse SQL di Azure Synapse, è disponibile una vasta gamma di strumenti leader nel settore tra cui scegliere. Questo articolo evidenzia le società partner di Microsoft che offrono strumenti e soluzioni per la gestione dei dati in grado di supportare Azure Synapse.

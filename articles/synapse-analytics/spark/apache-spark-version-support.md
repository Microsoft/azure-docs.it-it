# <a name="supported-language-and-runtime-versions-for-apache-spark-and-dependent-components"></a>Linguaggi e versioni di runtime supportati per Apache Spark e componenti dipendenti  

Il Apache Spark nel servizio di analisi di Azure sinapsi supporta diversi servizi e tempi di esecuzione diversi in questo documento sono elencate le versioni.

## <a name="spark-version"></a>Versione di Spark

Spark 2,4 è la versione attualmente supportata.

## <a name="operating-system-version"></a>Versione del sistema operativo

Apache Spark in sinapsi di Azure è in esecuzione nella versione 16,04 di Ubuntu.

## <a name="java-version"></a>Versione Java

Apache Spark in Azure sinapsi include Java version 1.8.0 _222

## <a name="scala-version"></a>Versione di Scala

Apache Spark in sinapsi di Azure include scala 2.11.12

## <a name="net-core-version"></a>Versione di .NET Core

Apache Spark in sinapsi di Azure include .NET Core 3,0

## <a name="net-support-for-apache-spark"></a>Supporto .NET per Apache Spark

La sinapsi di Azure include il supporto .NET per Apache Spark 0.5.0

## <a name="delta-lake"></a>Delta Lake

Sinapsi Analytics include il supporto per Delta Lake 0.5.0

## <a name="python-version"></a>Versione Python

Apache Spark in sinapsi di Azure include un'installazione di Python versione 3.6.1

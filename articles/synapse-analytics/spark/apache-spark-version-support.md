# <a name="supported-language-and-runtime-versions-for-apache-spark-and-dependent-components"></a>Versioni di linguaggio e runtime supportate per Apache Spark e componenti dipendenti  

Il servizio Apache Spark in Azure Synapse Analytics supporta diversi tempi di esecuzione e servizi in questo documento sono elencate le versioni.

## <a name="spark-version"></a>Versione Spark

Spark 2.4 è la versione attualmente supportata.

## <a name="operating-system-version"></a>Versione del sistema operativo

Apache Spark in Azure Synapse viene eseguito su Ubuntu versione 16.04.Apache Spark in Azure Synapse runs on Ubuntu version 16.04.

## <a name="java-version"></a>Versione Java

Apache Spark in Azure Synapse include la versione Java 1.8.0_222

## <a name="scala-version"></a>Versione di Scala

Apache Spark in Azure Synapse include Scala 2.11.12

## <a name="net-core-version"></a>Versione di .NET Core

Apache Spark in Azure Synapse include .NET Core 3.0

## <a name="net-support-for-apache-spark"></a>Supporto .NET per Apache Spark

Azure Synapse include il supporto .NET per Apache Spark 0.5.0Azure Synapse includes .NET Support for Apache Spark 0.5.0

## <a name="delta-lake"></a>Delta Lake

Synapse Analytics include il supporto per Delta Lake 0.5.0

## <a name="python-version"></a>Versione Python

Apache Spark in Azure Synapse include un'installazione di Python versione 3.6.1

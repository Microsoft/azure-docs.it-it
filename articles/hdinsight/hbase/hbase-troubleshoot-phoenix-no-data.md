# <a name="scenario-hdp-upgrade-causes-no-data-in-apache-phoenix-views-in-azure-hdinsight"></a>Scenario: HDP upgrade causes no data in Apache Phoenix views in Azure HDInsight

Questo articolo descrive i passaggi per la risoluzione dei problemi e le possibili soluzioni per i problemi relativi all'interazione con i cluster HDInsight di Azure.This article describes troubleshooting steps and possible resolutions for issues when interacting with Azure HDInsight clusters.

## <a name="issue"></a>Problema

Una vista Apache Phoenix non contiene alcuna data dopo l'aggiornamento da HDP 2.4 a HDP 2.5.

## <a name="cause"></a>Causa

La tabella dell'indice per le viste (tutti gli indici per le viste vengono archiviati in una singola tabella Apache HBase fisica) viene troncata durante l'aggiornamento

## <a name="resolution"></a>Risoluzione

Eliminare e ricreare tutti gli indici della vista dopo l'aggiornamento.

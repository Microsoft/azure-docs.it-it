# <a name="support-for-microsoft-commercial-marketplace"></a>Supporto per Microsoft Commercial Marketplace

Sono disponibili domande su come iniziare a usare Microsoft Commercial Marketplace Publisher? Ecco un elenco di opzioni di supporto per il Marketplace commerciale. Oltre alle risorse riportate di seguito, è anche possibile ottenere molte risposte alle domande nel [Forum della community del Marketplace di C + ai](https://www.microsoftpartnercommunity.com/t5/Marketplace/bd-p/2222).  

## <a name="onboarding"></a>Onboarding

Aprire un ticket con il [supporto](https://partner.microsoft.com/support/v2/?stage=1) per la pubblicazione in Microsoft Marketplace per i problemi relativi al caricamento e all'introduzione.

## <a name="partner-center"></a>Centro per i partner   

| Canale di supporto | Description | Disponibilità |  
|:--- |:--- |:--- |  
| Per assistenza, visitare la pagina Create an incident (Creare un evento imprevisto) disponibile nel [supporto per il Marketplace](https://partner.microsoft.com/support/v2/?stage=1)</li> </ul> | Supporto per il centro per i partner. | Viene fornito il supporto 24x5. |

# <a name="cloud-partner-portal-migration-to-partner-center"></a>Migrazione dal portale Cloud Partner al Centro per i partner

Per semplificare l'account e l'esperienza di gestione delle offerte, Microsoft sta eseguendo la migrazione delle offerte dal portale per i partner cloud al Centro per i partner. Se attualmente gestisci le garanzie di co-selling nel catalogo One Commercial Partner Go-to-Market (OCP GTM), sarai in grado di aggiornare presto le tue garanzie di co-selling nel Partner Center.

| **Tipo di offerta** | **Gestione delle offerte migrate dal portale per i partner cloud al Centro per i partner** | **Possibilità di aggiornare le garanzie di co-selling nel Centro per i partner** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| Software as a Service (SaaS) |     Ora disponibile       |   Disponibilità generale Aprile 2020      |
| Dynamics 365 per il coinvolgimento dei clienti & PowerApps      |     Ora disponibile        |   Ora disponibile      |
| Dynamics 365 for Operations     |     Ora disponibile       |   Ora disponibile     |
| Dynamics 365 Business Central      |     Ora disponibile        |   Disponibilità generale Aprile 2020     |
| Applicazione di Azure     |     Ora disponibile       |   Disponibilità generale Aprile 2020     |
| Macchina virtuale di Azure      |     Disponibilità generale aprile-maggio 2020       |   In attesa di essere annunciato    |
| Contenitore di AzureAzure Container     |     Disponibilità generale aprile-maggio 2020       |   In attesa di essere annunciato     |
| Servizio di consulenza      |     Disponibilità generale aprile-maggio 2020       |   In attesa di essere annunciato     |
| Modulo Edge IoT      |     Disponibilità generale aprile-maggio 2020       |   In attesa di essere annunciato     |
| App del servizio Power BI     |     Disponibilità generale aprile-maggio 2020       |   In attesa di essere annunciato     |

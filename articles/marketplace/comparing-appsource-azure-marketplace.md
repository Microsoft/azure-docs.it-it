# <a name="comparing-microsoft-appsource-and-azure-marketplace"></a>Confronto tra Microsoft AppSource e Azure Marketplace

Il marketplace commerciale Microsoft include [Microsoft AppSource](https://appsource.microsoft.com) e [Azure Marketplace.](https://azuremarketplace.microsoft.com) Ogni vetrina serve esigenze specifiche dei clienti e diversi segmenti di pubblico di destinazione, in modo da poter assicurarsi che la soluzione o il servizio sia disponibile per i clienti giusti.

## <a name="understanding-the-differences-between-storefronts"></a>Differenze tra le vetrine

Per scegliere una vetrina, identifica innanzitutto il pubblico di destinazione per la tua offerta. Se ci si rivolge ai destinatari di più vetrine, è possibile pubblicare una sola volta per la vendita in tutte le vetrine.

| Azure Marketplace | AppSource |
|:--- |:--- |
| Sviluppatori e professionisti IT | Utenti aziendali e decision maker aziendali |

Le tabelle seguenti descrivono i vantaggi che è possibile conseguire usando una vetrina:

| Vantaggi | Azure Marketplace | AppSource |
|:--- |:--- | :--- |
| **Flessibilità di fatturazione** | Azure Marketplace supporta i tipi di offerta gratuita, di valutazione, BYOL e transact. Le offerte Transact sono basate sul consumo o su sottoscrizione. Le offerte basate sul consumo vengono fatturate in base all'utilizzo orario e possono fornire un periodo gratuito iniziale. I servizi in abbonamento vengono fatturati mensilmente o annualmente a tariffa fissa o per posto.| AppSource supporta i tipi di offerta gratuita, di prova, BYOL e transact. È possibile vendere in modalità collaborativa a nuovi clienti. |

# <a name="commercial-marketplace-policies-and-terms"></a>Informative e condizioni del marketplace commerciale

Il _marketplace commerciale Microsoft è_ una piattaforma gestita o di proprietà di Microsoft, tramite la quale i clienti possono presentare o acquisire offerte. Se non diversamente specificato, il marketplace commerciale include Microsoft AppSource e Azure Marketplace.

Grazie per l'interesse per la pubblicazione di offerte nel marketplace commerciale. Microsoft si impegna a collaborare con l'utente per creare una ricca fonte di soluzioni cloud e una raccolta di offerte aziendali che soddisfano i clienti di tutto il mondo e consentono di creare la propria azienda.

Le offerte nel marketplace commerciale devono essere conformi ai criteri e ai termini. Le aggiorneremo di tanto in tanto per garantire una buona esperienza del cliente e aiutare i partner a riuscire. È possibile inviare commenti e suggerimenti sui criteri o sulle condizioni nel forum Microsoft AppSource [e Azure Marketplace.](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/bd-p/2222)

## <a name="publisher-agreement"></a>Contratto Publisher

- [Contratto Microsoft Publisher](https://go.microsoft.com/fwlink/?LinkID=699560)
- [Cronologia delle modifiche per il Contratto di Microsoft Publisher](https://go.microsoft.com/fwlink/?linkid=2159975&clcid=0x409)

Il Contratto Microsoft Publisher descrive la relazione per la pubblicazione di offerte nel marketplace commerciale. Controlla l'accesso e l'uso delle funzionalità Partner Center correlate alla pubblicazione e all'inserzione di offerte nei negozi online del marketplace commerciale.

# <a name="azure-marketplace-listing-guidelines"></a>Linee guida per la presentazione in Azure Marketplace

Questo documento contiene linee guida relative ai requisiti e un elenco di controllo per i revisori di nuove offerte e nuovi servizi presentati in Azure Marketplace.

Tutte le offerte devono soddisfare [tutti i requisiti di presentazione](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#all-listing-requirements) indicati di seguito. Esistono requisiti ed elenchi di controllo aggiuntivi per tipi di presentazione specifici: offerte con [ versioni di prova](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#trial-offer-requirements) allegate, [app SaaS](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#saas-app-requirements), [contenitori](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#container-offer-requirements) e [offerte di consulenza](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#consulting-offer-requirements).

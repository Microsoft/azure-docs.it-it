# <a name="iot-edge-module-offer"></a>Offerta di modulo IoT Edge

Il [portale cloud partner](https://cloudpartner.azure.com/) non supporta più la creazione e la gestione delle offerte del modulo IOT Edge. Questa funzionalità è stata spostata nel [centro](https://partner.microsoft.com/pcv/)per i partner Microsoft. Per altre informazioni, vedere [creare, configurare e pubblicare un'offerta di modulo IOT Edge in Azure Marketplace](../../partner-center-portal/azure-iot-edge-module-creation.md).

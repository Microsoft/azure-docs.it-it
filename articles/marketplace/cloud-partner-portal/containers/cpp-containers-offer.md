# <a name="containers"></a>Contenitori

Il [portale Cloud Partner](https://cloudpartner.azure.com/) non supporta più la creazione e la gestione delle offerte di contenitori di Azure. Questa funzionalità è stata spostata nel [Centro per i partner](https://partner.microsoft.com/pcv/) Microsoft. Per altre informazioni, vedere [Creare un'offerta per il contenitore di Azure](../../partner-center-portal/create-azure-container-offer.md).

# <a name="preferred-solutions-in-azure-marketplace-and-appsource"></a>Soluzioni preferenziali in Azure Marketplace e AppSource

Una soluzione preferenziale Microsoft è un'applicazione cloud selezionata per la qualità, le prestazioni e la capacità di soddisfare le esigenze dei clienti in un determinato settore verticale o un'area della soluzione. Un team di esperti Microsoft convalida le soluzioni di partner con competenze e capacità specifiche comprovate. Queste soluzioni sono presenti nelle vetrine del marketplace cloud, Azure Marketplace e AppSource, nonché nel portale di Azure.These solutions are featured in our cloud marketplace storefronts, Azure Marketplace, and AppSource, as well as in the Azure portal. Le soluzioni preferite su AppSource possono essere individuate dai verticali del settore. Le soluzioni preferite in Azure Marketplace e nel portale di Azure possono essere individuate tra categorie di soluzioni orizzontali.

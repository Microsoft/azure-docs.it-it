# <a name="jenkins-plugins-for-azure"></a>Plug-in Jenkins per Azure

I plug-in Jenkins seguenti supportano varie funzionalità per l'uso con Azure.The following Jenkins plugins support various features for use with Azure.
  
| Plug-in Jenkins | Descrizione                                   |
|------------------------------------------------------------------------------|
| [Plug-in Servizio app di Azure](https://plugins.jenkins.io/azure-app-service)     | Plug-in Jenkins per distribuire un servizio app di Azure (attualmente supporta solo app Web). | 
| [Plug-in di Azure AD](https://plugins.jenkins.io/azure-ad)                       | Plug-in Jenkins che supporta l'autenticazione & l'autorizzazione tramite Azure Active Directory. | 
| [Plug-in Gestione artefatti di AzureAzure Artifact Manager plugin](https://plugins.jenkins.io/azure-artifact-manager) | Azure Artifact Manager plugin is an Artifact Manager that allows you store your artifacts into Azure Blob Storage. Il plug-in Azure Artifact Manager funziona in modo trasparente con Jenkins e i processi, è simile a Gestione artefatto predefinito. | 
| [Plug-in degli Agenti del servizio contenitore di Azure](https://plugins.jenkins.io/azure-container-agents) | Azure Container Agents plugin can help you to run a container as an agent in Jenkins | 
| [Plug-in Attività del Registro di sistema del contenitore di AzureAzure Container Registry](https://plugins.jenkins.io/azure-container-registry-tasks)       | Plug-in Jenkins per inviare una richiesta docker-build al Registro di sistema [del contenitore](/azure/container-registry/container-registry-tasks-overview)di Azure. |
| [Plug-in del servizio Azure Container](https://plugins.jenkins.io/azure-acs)       | Plug-in Jenkins per distribuire le configurazioni nel servizio contenitore di Azure (AKS). | 
| [Plug-in di Credenziali di Azure](https://plugins.jenkins.io/azure-credentials)      | Plug-in Jenkins per gestire le credenziali di Azure.Jenkins plugin to manage Azure credentials. | 
| [Plug-in di Funzioni di Azure](https://plugins.jenkins.io/azure-function)           | Jenkins plugin to deploy an Azure Function. | 
| [Plug-in di Azure IoT EdgeAzure IoT Edge plugin](https://plugins.jenkins.io/azure-iot-edge)           | Azure IoT Edge plugin makes it easy to set up a CI/CD pipeline for developing an IoT Edge solution on Jenkins. | 
| [Plug-in di Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Plug-in Jenkins per progetti Linux Azure Service Fabric. |
| [Plug-in di Archiviazione di Azure](https://plugins.jenkins.io/windows-azure-storage)     | plug-in per il caricamento di elementi di compilazione o il download delle dipendenze di compilazione dall'archiviazione BLOB di Microsoft Azure.Plugin for uploading build artifacts to, or downloading build dependencies from, Microsoft Azure Blob storage. | 
| [Plug-in degli agenti di macchina virtuale di AzureAzure VM agents plugin](https://plugins.jenkins.io/azure-vm-agents)         | Plug-in Jenkins per creare agenti Jenkins nelle macchine virtuali di Azure (tramite il modello di Azure Resource Manager). | 
| [Plug-in del set di scalabilità delle macchine virtuali di AzureAzure virtual machine scale set plugin](https://plugins.jenkins.io/azure-vmss)           | Plug-in Jenkins per distribuire le immagini delle macchine virtuali nei set di scalabilità delle macchine virtuali di Azure.Jenkins plugin to deploy VM images to Azure virtual machine scale sets. |

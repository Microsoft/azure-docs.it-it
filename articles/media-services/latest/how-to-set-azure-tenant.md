# <a name="how-to-find-your-azure-tenant-id"></a>Come trovare l'ID tenant di Azure

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

[!INCLUDE [quickstarts-free-trial-note](../../../includes/quickstarts-free-trial-note.md)]

## <a name="find-your-tenant-id-in-the-portal"></a>Trovare l'ID tenant nel portale

1. Accedere al [portale di Azure](https://portal.azure.com).
1. Selezionare l'icona Azure Active Directory in servizi di Azure. Viene visualizzata la pagina Active Directory.
1. L'ID tenant si trova in **informazioni di base**.
1. Copiare l'ID tenant e incollarlo in un documento di testo di propria scelta per usarlo in seguito.

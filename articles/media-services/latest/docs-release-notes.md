# <a name="azure-media-services-v3-documentation-updates"></a>Aggiornamenti della documentazione di servizi multimediali di Azure V3

>Quando si torna a visitare questa pagina per informazioni sugli aggiornamenti, è possibile ricevere notifiche copiando e incollando l'URL `https://docs.microsoft.com/api/search/rss?search=%22Azure+Media+Services+v3+documentation+updates%22&locale=en-us` nel lettore di feed RSS.

Questo articolo illustra gli aggiornamenti più recenti della documentazione di servizi multimediali V3.

## <a name="june-2020"></a>Giugno 2020

* L'anteprima di analisi video in tempo reale su IoT Edge è stata pubblica. Per informazioni dettagliate, vedere la documentazione relativa a [IOT Edge video in tempo reale](https://docs.microsoft.com/azure/media-services/live-video-analytics-edge/) .
* Nuove guide introduttive:

    * [Usare il portale per caricare, codificare ed eseguire lo streaming di contenuto](manage-assets-quickstart.md)
    * [Usare il portale per crittografare contenuto](encrypt-content-quickstart.md)

## <a name="april-2020"></a>Aprile 2020

* È stata eseguita la migrazione della documentazione di Azure Media Player alla [documentazione di Azure](../azure-media-player/azure-media-player-overview.md).
* È stata aggiunta la Guida introduttiva [streaming live con Open Broadcasting Studio (OBS)](live-events-obs-quickstart.md) . Viene illustrato come creare un flusso live di servizi multimediali usando il portale di Azure e l'OBS.

## <a name="march-2020"></a>Marzo 2020

È stata aggiunta la Guida introduttiva [streaming live con Telestream Wirecast](live-events-wirecast-quickstart.md) . Viene illustrato come creare un flusso live di servizi multimediali di Azure usando il portale di Azure e Telestream Wirecast.

## <a name="citrix"></a>Citrix

![Logo Citrix](./media/partners/citrix.png)

L'IT è in grado di ottimizzare i costi con strumenti di scalabilità intelligenti, offrendo al contempo un'esperienza utente straordinaria e testata sul campo rispetto alle applicazioni più difficili di tutti i settori. Inoltre, Citrix Managed Desktops è un programma desktop as-a-service abilitato per Windows Virtual Desktop che fornisce una soluzione di gestione semplice e basata su cloud per la distribuzione di app e desktop virtuali a qualsiasi dispositivo.

## <a name="vmware"></a>VMware

![VMware Logo](./media/partners/vmware.png)

## <a name="cloudhouse"></a>Casa nuvolosa

## <a name="thinprint"></a>Stampa sottile

![ThinPrint Logo](./media/partners/thinprint.png)

ThinPrint è un provider di servizi a valore aggiunto di Windows Virtual Desktop che offre una stampa cloud semplice e sicura da Windows Virtual Desktop. Con i suoi servizi e software, l'infrastruttura di stampa esistente può essere utilizzata per stampare documenti dal cloud. ThinPrint consente la connessione a stampanti locali e di rete, semplificando la stampa in ufficio o in remoto.

La soluzione ezeep di ThinPrint estende il valore di Windows Virtual Desktop abilitando la connessione all'infrastruttura di stampa aziendale esistente. ezeep offre agli utenti il controllo sulla stampa nell'azienda, indipendentemente da dove si trovino. Utilizzando ezeep, gli utenti possono colmare il divario tra Windows Virtual Desktop e l'hardware di stampa.

- [Vai al sito web del partner](https://www.ezeep.com/wvd-printing).

## <a name="tricerat"></a>Tricerat

![Tricerat Logo](./media/partners/tricerat.png)

Tricerat offre una soluzione di gestione della stampa superiore per Windows Virtual Desktop e altre piattaforme desktop. Il software Tricerat dispone di funzionalità solide, che offrono un'esperienza migliore sia per gli utenti che per gli amministratori. Gli amministratori ottengono efficienza grazie alla gestione completa dei driver, alla distribuzione semplificata delle code di stampa e a una gestione coerente su piattaforme ibride. L'esperienza utente migliora con tempi di accesso più brevi, code di stampa intelligenti basate su utente, dispositivo e percorso di rete e opzioni self-service per la selezione rapida della stampante. 

Con Tricerat, la stampa è senza soluzione di continuità in Windows Virtual Desktop e oltre. Il software Tricerat consente agli amministratori di collegare facilmente le stampanti locali al cloud, estendendo la gestione della stampa aziendale dagli ambienti tradizionali a nuovi spazi di lavoro moderni.

- [Visitare il sito Web del partner](https://www.tricerat.com/microsoft-printing).

## <a name="workspot"></a>Area di lavoro

# <a name="azure-security-center-for-iot-security-alerts"></a>Avvisi di sicurezza del Centro sicurezza di Azure per IoT

Il Centro sicurezza di Azure per l'IoT analizza continuamente la soluzione IoT usando analisi avanzate e informazioni sulle minacce per avvisare l'utente di attività dannose.
Inoltre, è possibile creare avvisi personalizzati in base alle proprie conoscenze sul comportamento previsto del dispositivo.
Un avviso funge da indicatore di un potenziale compromesso e deve essere studiato e corretti.

In questo articolo troverai un elenco di avvisi incorporati che possono essere attivati sul tuo hub IoT e/o sui dispositivi IoT.
Oltre agli avvisi predefiniti, Centro sicurezza di Azure per IoT consente di definire avvisi personalizzati in base al comportamento previsto dell'hub IoT e/o del dispositivo.
Per ulteriori dettagli, vedere [Avvisi personalizzabili](concept-customizable-security-alerts.md).

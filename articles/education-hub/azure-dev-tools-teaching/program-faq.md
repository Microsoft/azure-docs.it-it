# <a name="frequently-asked-questions-about-the-education-hub"></a>Domande frequenti sull'hub di Education

Questo articolo contiene le risposte alle domande frequenti su Azure for Students, Azure for Students Starter e Azure Dev Tools for Teaching.

## <a name="azure-for-students"></a>Azure for Students

### <a name="what-happens-after-i-use-my-100-credit-or-im-at-the-end-of-12-months"></a>Che cosa succede una volta esaurito il credito di $ 100 o al termine dei 12 mesi?

Dopo aver esaurito il credito disponibile o aver raggiunto il termine dei 12 mesi, la sottoscrizione di Azure viene disabilitata. Per continuare a usare [Azure](https://account.azure.com/), è possibile eseguire l'aggiornamento a una sottoscrizione con pagamento in base al consumo nel portale degli account Azure. Per informazioni sulle condizioni per l'utilizzo di Azure for Students, vedere le [condizioni dell'offerta](https://azure.microsoft.com/offers/ms-azr-0170p/).

### <a name="can-i-get-azure-for-students-again-next-year"></a>Sarà possibile usare di nuovo Azure for Students l'anno successivo?

Grazie per l'interesse dimostrato in Azure for Students. Siamo lieti che il servizio sia stato apprezzato. Al momento, non sono previsti rinnovi. Ricontrollare in seguito per aggiornamenti.

### <a name="what-are-subscriptions-and-how-do-they-relate-to-azure-for-students"></a>Che cosa sono le sottoscrizioni e come sono correlate ad Azure for Students?

Le sottoscrizioni forniscono l'accesso ai servizi di Azure. Azure for Students offre un credito di $ 100 per 12 mesi. Include l'accesso a più di 25 servizi gratuiti, tra cui risorse di calcolo, rete, archiviazione e database. Eventuali addebiti sostenuti durante questo periodo vengono dedotti dal credito. Per continuare a usare i servizi di Azure al termine dei 12 mesi o dopo aver esaurito il credito di $ 100, a seconda di quale situazione si verifichi per prima, è necessario eseguire l'aggiornamento a una sottoscrizione con pagamento in base al consumo.

### <a name="what-happens-with-my-azure-services-if-i-dont-upgrade"></a>Che cosa succede ai servizi di Azure se non si effettua l'aggiornamento?

Se si decide di non effettuare l'aggiornamento al termine dei 12 mesi o dopo aver esaurito il credito di $ 100, a seconda di quale delle due situazioni si verifichi per prima, tutti i prodotti distribuiti verranno rimossi e non sarà possibile accedervi. A partire dal termine della sottoscrizione gratuita, si hanno a disposizione 90 giorni per eseguire l'aggiornamento a una sottoscrizione con pagamento in base al consumo.

### <a name="how-do-i-know-how-much-of-the-100-credit-i-have-left"></a>Come si può sapere qual è l'importo residuo del credito di $ 100?

È possibile verificare il credito rimanente nel [portale Azure Sponsorships](https://www.microsoftazuresponsorships.com/).

### <a name="how-do-i-download-the-software-developer-tools"></a>Come è possibile scaricare gli strumenti di sviluppo software?

La sottoscrizione di Azure for Students offre l'accesso a determinati strumenti di sviluppo software. Per accedere e scaricare gli strumenti di sviluppo software, è necessario avere una sottoscrizione di Azure for Students attiva e aggiornata. Per scaricare gli strumenti di sviluppo software usando la sottoscrizione di Azure for Students, visitare l'[hub di Education](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software).

### <a name="what-is-microsoft-learn"></a>Che cos'è Microsoft Learn?

[Microsoft Learn](https://docs.microsoft.com/learn/) è una piattaforma gratuita di formazione online che consente di apprendere le tecnologie di Azure in base ai propri ritmi. I percorsi di apprendimento combinano moduli che consentono di iniziare con le nozioni di base, per poi passare a metodi avanzati che affrontano problematiche reali.

### <a name="can-azure-for-students-be-used-for-production-or-only-for-development"></a>È possibile usare Azure for Students solo per gli ambienti di sviluppo o anche per quelli di produzione?

Azure for Students fornisce l'accesso a tutti i prodotti di Azure espressamente destinati a sostenere l'istruzione o l'insegnamento, la ricerca non commerciale o le attività di progettazione, sviluppo, test e dimostrazione di applicazioni software per gli scopi indicati sopra.

### <a name="can-i-apply-any-of-my-100-credit-toward-azure-marketplace-offers"></a>È possibile usare il credito di $ 100 per le offerte di Azure Marketplace?

No. Non è possibile applicare il credito alle offerte di Azure Marketplace. Tuttavia, molti partner di Azure Marketplace offrono versioni di valutazione gratuite o piani di livello gratuito per le loro soluzioni.

## <a name="azure-for-students-starter"></a>Azure for Students Starter

### <a name="what-is-azure-for-students-starter"></a>Che cos'è Azure for Students Starter?

Azure for Students Starter permette di iniziare a usare i prodotti di Azure che servono per sviluppare sul cloud. Non sono previsti costi. Questo vantaggio consente di accedere a un livello gratuito dei servizi seguenti:

- Servizio app di Azure
- Funzioni di Azure
- Hub di notifica di Azure
- Database di Azure per MySQL
- Application Insights
- Azure DevOps Services (in precedenza Visual Studio Team Services)

Azure for Students Starter è disponibile gratuitamente per gli studenti verificati e senza impegno o limite di tempo. Per informazioni dettagliate sulle condizioni per l'utilizzo, vedere l'[offerta Azure for Students Starter](https://azure.microsoft.com/offers/ms-azr-0144p/).

### <a name="who-is-eligible-for-azure-for-students-starter"></a>Chi sono gli utenti idonei per Azure for Students Starter?

Azure for Students Starter è disponibile solo per gli studenti che soddisfano i requisiti seguenti:

- Se si risiede negli Stati Uniti, è necessario affermare di avere almeno 13 anni di età.
- Se si risiede in paesi diversi da Stati Uniti, è necessario affermare di avere almeno 16 anni di età.
- È necessario verificare il proprio stato accademico tramite l'indirizzo di posta elettronica dell'organizzazione. È anche possibile usare Shibboleth, se supportato dall'organizzazione.

Questa offerta non è disponibile per l'uso in corsi MOOC (Massive Open Online Course) o in altri corsi professionali di formazione offerti da organizzazioni a fini di lucro.

Questa offerta è limitata a una sottoscrizione di Azure for Students Starter per ogni cliente idoneo. Non è trasferibile e non è cumulabile con altre offerte, se non diversamente autorizzato da Microsoft.

### <a name="will-i-have-to-pay-something-at-any-point-in-time"></a>A un certo momento, sarà necessario pagare?

Per l'offerta Azure for Students Starter non è richiesta una carta di credito. L'offerta fornisce l'accesso a un set limitato di servizi di Azure. Tuttavia, in qualsiasi momento, è possibile eseguire l'aggiornamento a una sottoscrizione con pagamento in base al consumo per ottenere l'accesso a tutti i servizi di Azure.

### <a name="how-do-i-download-the-software-developer-tools"></a>Come è possibile scaricare gli strumenti di sviluppo software?

La sottoscrizione di Azure for Students offre l'accesso a determinati strumenti di sviluppo software, disponibili gratuitamente per il download.

Per accedere agli strumenti di sviluppo software, è necessario avere una sottoscrizione di Azure for Students attiva e aggiornata.

È possibile scaricare il software nell'[hub di Education](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software).

### <a name="what-is-microsoft-learn"></a>Che cos'è Microsoft Learn?

[Microsoft Learn](https://docs.microsoft.com/learn/) è una piattaforma gratuita di formazione online che consente di apprendere le tecnologie di Azure in base ai propri ritmi. I percorsi di apprendimento combinano moduli che consentono di iniziare con le nozioni di base, per poi passare a metodi avanzati che affrontano problematiche reali.

## <a name="azure-academic-grant"></a>Azure Academic Grant

### <a name="how-do-i-start-using-my-azure-course-credits"></a>Come si fa a iniziare a usare i crediti dei corsi di Azure?

Per accedere ai crediti dei corsi di Azure, è possibile creare una nuova sottoscrizione di Microsoft Azure Academic Grant. Selezionare il pulsante **Attiva** nel messaggio di posta elettronica di approvazione della sponsorizzazione.

È anche possibile convertire una sottoscrizione esistente nell'offerta Microsoft Azure Sponsorship per accedere ai crediti. Informazioni dettagliate su come convertire la sottoscrizione sono disponibili nella domanda successiva.

### <a name="can-i-associate-my-course-credits-with-an-existing-subscription"></a>È possibile associare i crediti dei corsi a una sottoscrizione esistente?

È possibile associare i crediti dei corsi a una sottoscrizione esistente nell'account autorizzato all'offerta. Per associare i crediti dei corsi, contattare il [supporto di Azure](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview).

### <a name="why-do-i-see-a-0-balance-in-the-azure-portal"></a>Perché nel portale di Azure è visualizzato un saldo di $ 0?

Quando si accede ai dettagli della sottoscrizione nel portale di Azure, viene visualizzato $ 0,00. L'offerta prevede uno sconto del 100% per tutti i servizi. Il portale mostra quindi l'importo che verrà addebitato durante il periodo mensile di utilizzo, che dovrà essere $ 0,00.

Per visualizzare informazioni sul saldo e sulla sponsorizzazione, passare ad [Azure Sponsorship](https://www.microsoftazuresponsorships.com/balance) e accedere al proprio account.

### <a name="can-i-apply-my-course-credits-to-an-existing-enterprise-agreement-ea"></a>È possibile applicare i crediti dei corsi a un contratto Enterprise Agreement esistente?

Non è possibile associare l'offerta dei crediti dei corsi alle sottoscrizioni di account con contratto Enterprise.

Per applicare i crediti dei corsi, è necessario creare un nuovo account al di fuori del contratto Enterprise, che potrà quindi essere autorizzato.

Al termine del periodo di sponsorizzazione, è possibile associare tale sottoscrizione nuovamente al contratto Enterprise.

> [!WARNING]
> Se si associa l'account al contratto Enterprise prima della fine della sponsorizzazione, tutti i fondi della sponsorizzazione verranno terminati. Per altre informazioni, vedere le condizioni per l'utilizzo dell'[offerta Azure Sponsorship](https://azure.microsoft.com/offers/ms-azr-0143p/).

### <a name="can-i-apply-my-course-credits-to-my-csp-subscription"></a>È possibile applicare i crediti dei corsi alla sottoscrizione CSP?

No, non è possibile associare l'offerta dei crediti dei corsi a sottoscrizioni Cloud Solution Provider (CSP).

### <a name="can-i-move-my-course-credits-to-another-account"></a>È possibile spostare i crediti dei corsi in un altro account?

Sì, è possibile spostare l'entitlement dei crediti dei corsi in un altro account. A tale scopo, contattare il [supporto di Azure](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview).

### <a name="are-azure-marketplace-applications-covered-by-course-credits"></a>Le applicazioni di Azure Marketplace sono coperte dai crediti dei corsi?

No, le applicazioni di terze parti non sono coperte da crediti dei corsi. I relativi costi verranno addebitato sulla carta di credito associata all'account. Per altre informazioni:

- Visualizzare i servizi del marketplace in [Azure Marketplace](https://azuremarketplace.microsoft.com/marketplace/).
- Consultare le condizioni per l'utilizzo dell'[offerta Azure Sponsorship](https://azure.microsoft.com/offers/ms-azr-0143p/).

### <a name="i-have-a-previous-balance-due-can-i-pay-it-off-with-my-course-credits"></a>Nel caso di un saldo dovuto precedente, è possibile pagarlo con i crediti dei corsi?

I crediti dei corsi di Azure coprono l'utilizzo solo dal momento in cui si attiva l'offerta Azure Sponsorship. L'utente è responsabile di tutti gli addebiti precedenti alla data di inizio dell'offerta.

### <a name="how-do-i-know-if-my-subscription-is-on-the-azure-course-credit-offer"></a>Come si può sapere se l'offerta relativa ai crediti dei corsi di Azure è attivata per la sottoscrizione?

Se si esamina una sottoscrizione specifica nel pannello Sottoscrizioni del portale di Azure, si vedrà **Nome offerta** come una delle proprietà. Il nome dell'offerta sarà **Azure Sponsorship**, se è connessa ai crediti dei corsi. Se l'offerta **Azure Sponsorship** non viene visualizzata, contattare il supporto tecnico per convertirla.

### <a name="how-do-i-download-software"></a>Come si può scaricare il software?

La sottoscrizione di Microsoft Azure Dev Tools for Teaching offre l'accesso a determinati strumenti di sviluppo software. Questi strumenti sono disponibili gratuitamente per il download.

È possibile scaricare il software nella [sezione software dell'hub di Education](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software).

### <a name="how-do-we-distribute-software-to-our-students"></a>Come si distribuisce il software agli studenti?

L'istituto di istruzione, in qualità di sottoscrittore di Azure Dev Tools for Teaching, ottiene l'accesso allo store dell'hub di Education. Gli studenti accedono ai servizi cloud nello [store dell'hub di Education](https://azureforeducation.microsoft.com/devtools), disponibile nel [portale di Azure](https://portal.azure.com/).

Gli studenti accedono al portale di Azure con le credenziali dell'istituto di istruzione o di Azure Dev Tools for Teaching. Quindi, gli studenti aprono lo store dell'hub di Education e accedono ai download di software disponibili.

### <a name="is-azure-dev-tools-for-teaching-available-internationally"></a>L'offerta Azure Dev Tools for Teaching è disponibile a livello internazionale?

Sì, è disponibile in più di 140 paesi/aree in cui Azure è disponibile in commercio.

### <a name="which-languages-are-available-in-the-software-to-end-users"></a>Quali lingue sono disponibili nel software per gli utenti finali?

Lo store dell'hub di Education è disponibile nelle lingue seguenti: arabo, cinese semplificato, cinese tradizionale, coreano, danese, ebraico, francese, giapponese, inglese, italiano, olandese, portoghese, russo, spagnolo, svedese, tedesco e turco.

### <a name="if-our-students-download-software-through-the-azure-dev-tools-for-teaching-program-do-they-get-unlimited-use-of-the-software"></a>Se gli studenti scaricano il software tramite il programma Azure Dev Tools for Teaching, possono usufruirne di un uso illimitato?

Sì. Gli studenti ricevono un utilizzo illimitato del software per approfondire le loro attività di apprendimento e ricerca.

### <a name="if-our-students-create-viable-apps-and-products-using-azure-dev-tools-for-teaching-software-can-they-sell-them-commercially"></a>Se gli studenti creano app e prodotti fattibili con il software di Azure Dev Tools for Teaching, possono venderli in commercio?

Generalmente, no. Gli studenti non possono vendere app e prodotti creati con Azure Dev Tools for Teaching. Tuttavia, grazie a una partnership tra i team di Azure Dev Tools for Teaching e di Microsoft Store, gli studenti possono creare giochi e applicazioni da vendere in Windows Store.

### <a name="do-i-have-unlimited-use-of-the-software-through-the-azure-dev-tools-for-teaching-program"></a>Il software ottenuto tramite il programma Azure Dev Tools for Teaching è disponibile per un uso illimitato?

Sì. Se un membro del corpo docente si è registrato in un corso approvato, è idoneo per installare il software di Azure Dev Tools for Teaching nel proprio computer personale per uso non commerciale.

### <a name="how-do-i-access-my-visual-studio-enterprise-benefit"></a>Come si accede al vantaggio Visual Studio Enterprise?

L'amministratore della sottoscrizione di Azure Dev Tools for Teaching può ottenere l'accesso alla propria sottoscrizione di Visual Studio Enterprise richiedendolo tramite il [portale di gestione di Azure Dev Tools for Teaching](https://azureforeducation.microsoft.com/account/Subscriptions).

Una volta ricevuta l'approvazione, sarà possibile accedere al portale di [Visual Studio ](https://my.visualstudio.com/) e riscattare ulteriori vantaggi.

### <a name="does-microsoft-azure-dev-tools-for-teaching-include-microsoft-office"></a>Microsoft Azure Dev Tools for Teaching include Microsoft Office?

No. No. L'obiettivo di Microsoft Azure Dev Tools for Teaching è quello di fornire a dipartimenti, docenti e studenti gli strumenti necessari per espandere in modo specifico lo studio di sviluppo e test del software. Vengono pertanto fornite tecnologie come Windows Server, Visual Studio .NET, SQL Server e Platform SDK.

### <a name="does-azure-dev-tools-for-teaching-include-azure-credit"></a>Azure Dev Tools for Teaching include il credito Azure?

No, la sottoscrizione di Microsoft Azure Dev Tools for Teaching non include il credito Azure. Ma è possibile iscriversi ad Azure for Students, che offre $ 100 di credito Azure da usare per pagare i servizi di Azure. Vedere [Iniziare a costruirsi un futuro con Azure for Students!](https://aka.ms/student) per altre informazioni.

### <a name="do-students-need-an-office-365-or-active-directory-account-to-access-azure-dev-tools-for-teaching"></a>Gli studenti hanno bisogno di un account di Office 365 o di Active Directory per accedere ad Azure Dev Tools for Teaching?

No. Gli studenti non necessitano di un account di Office 365. Se hanno accesso all'account di Active Directory, usano le stesse credenziali per accedere al software.

Se non usano Active Directory, devono creare un [account Microsoft](https://account.microsoft.com/account) (se non ne hanno già uno) con lo stesso indirizzo di posta elettronica che hanno ricevuto.

### <a name="why-arent-my-sign-in-credentials-recognized-when-i-sign-in-to-azure-dev-tools-for-teaching"></a>Perché le credenziali di accesso non vengono riconosciute quando si accede ad Azure Dev Tools for Teaching?

Assicurarsi di accedere ad Azure Dev Tools for Teaching con le credenziali dell'istituto di istruzione. Potrebbe essere utile aprire una sessione di InPrivate Browsing.

Se non si è ancora in grado di accedere, contattare l'amministratore della sottoscrizione. Per trovare l'amministratore della sottoscrizione, [contattare Microsoft](https://aka.ms/adt4tsupport).

### <a name="how-do-i-find-my-subscriber-id"></a>Dove si trova l'ID sottoscrittore?

- **Se ci si registra per la prima volta nel programma**: il numero di ID sottoscrittore si trova nel messaggio di posta elettronica di benvenuto che si riceve.
- **Se la sottoscrizione è stata rinnovata**: l'ID sottoscrittore si trova nel messaggio di posta elettronica di rinnovo ricevuto dall'amministratore della sottoscrizione.

L'ID sottoscrittore è anche disponibile nel portale Sottoscrizione di Visual Studio. Dopo aver eseguito l'accesso, controllare in **Sottoscrizioni personali** nella pagina **Account personale**.

Se è necessaria assistenza per individuare l'ID sottoscrittore, [contattare Microsoft](https://azureforeducation.microsoft.com/institutions/Contact).

### <a name="are-we-automatically-enrolled-in-azure-dev-tools-for-teaching-if-we-receive-it-as-part-of-our-academic-volume-licensing-agreement"></a>La registrazione ad Azure Dev Tools for Teaching viene eseguita automaticamente se si riceve come parte di un contratto multilicenza accademico?

No, Microsoft non esegue automaticamente la registrazione se si ha un contratto multilicenza accademico. Il contratto multilicenza accademico include:

- Enrollment for Education Solutions (EES)
- Open Value Subscription Agreement for Education Solutions (OVS-ES)
- Campus Agreement
- School Agreement

È necessario registrarsi ad Azure Dev Tools for Teaching usando i codici promozionali appropriati disponibili nel messaggio di posta elettronica di benvenuto che si riceve per il contratto multilicenza accademico.

È anche necessario rinnovare la sottoscrizione alla scadenza. Il rinnovo non è automatico.

Se non si riesce a trovare il codice promozionale, [contattare Microsoft](https://azureforeducation.microsoft.com/institutions/Contact).

### <a name="how-and-when-do-we-renew-our-azure-dev-tools-for-teaching-subscription"></a>Come e quando si rinnova la sottoscrizione di Azure Dev Tools for Teaching?

60 giorni prima della scadenza della sottoscrizione, si inizieranno a ricevere promemoria tramite posta elettronica per rinnovarla. Nel messaggio di posta elettronica che si riceve selezionare il [collegamento per il rinnovo](https://portal.azureforeducation.microsoft.com/).

Se non si ricevono questi messaggi di promemoria e si ritiene che la sottoscrizione stia per scadere, [contattare Microsoft](https://aka.ms/adt4tsupport).

In alternativa, passare alla pagina di [rinnovo di Azure Dev Tools for Teaching](https://portal.azureforeducation.microsoft.com/) e completare la procedura.

Per verificare la data di scadenza della sottoscrizione, accedere al [portale di gestione di Azure Dev Tools for Teaching](https://azureforeducation.microsoft.com/account/Subscriptions) e controllare in **Sottoscrizioni**.

### <a name="what-if-i-need-more-help"></a>A chi è possibile rivolgersi per ulteriore assistenza?

[Contattare Microsoft](https://azureforeducation.microsoft.com/institutions/Contact) passando alla pagina del supporto per le sottoscrizioni e individuando la propria area.

### <a name="where-is-the-azure-dev-tools-for-teaching-privacy-and-cookies-policy"></a>Dove sono le informative di Azure Dev Tools for Teaching per la privacy e i cookie?

L'[informativa sulla privacy Microsoft](https://privacy.microsoft.com/PrivacyStatement) descrive quali dati personali vengono raccolti da Microsoft, come vengono elaborati e dove vengono condivisi.

Questa informativa sulla privacy copre un'ampia gamma di prodotti Microsoft, tra cui app, dispositivi, server, servizi, software e siti Web. Fornisce inoltre informazioni specifiche per i prodotti e dettagli sui criteri per l'uso dei cookie.

# <a name="help-students-download-software-developer-tools-from-the-azure-education-hub"></a>Aiuta gli studenti a scaricare gli strumenti di sviluppo software dal hub di Education di Azure

Tramite il programma Azure Dev Tools for Teaching, gli studenti che hanno una sottoscrizione di Azure for Students attiva e aggiornata possono scaricare gratuitamente specifici strumenti di sviluppo software. Azure for Students fornisce l'accesso a tutti i prodotti di Azure espressamente destinati a sostenere l'istruzione o l'insegnamento, la ricerca non commerciale o le attività di progettazione, sviluppo, test e dimostrazione di applicazioni software per gli scopi indicati sopra. Gli studenti scaricano gli strumenti dall'hub di Education di Azure.

## <a name="instructions-for-students-how-to-download-software-developer-tools"></a>Istruzioni per gli studenti: come scaricare gli strumenti di sviluppo software

Fare in modo che gli studenti seguano questa procedura per scaricare gli strumenti di sviluppo software:

1. Aprire un browser e passare alla [Hub di Education di Azure](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software).
1. Accedere con le credenziali di sottoscrizione di Azure per studenti.
1. Selezionare uno degli strumenti di sviluppo software nella hub di Education per aprire il riquadro di download.
1. Leggere le informazioni sul software per assicurarsi che il software sia compatibile con il computer.
1. Dopo aver confermato che il software è compatibile con il sistema, selezionare **Scarica** .
1. Seguire le istruzioni visualizzate per scaricare e installare il software.

   Azure for Students offre un credito di $ 100 per 12 mesi. Include l'accesso a più di 25 servizi gratuiti, tra cui risorse di calcolo, rete, archiviazione e database. Eventuali addebiti sostenuti durante questo periodo vengono dedotti dal credito.

# <a name="azure-for-students"></a>Azure for Students

Tramite il programma Azure Dev Tools for Teaching, gli studenti che hanno una sottoscrizione di Azure for Students attiva e aggiornata possono scaricare gratuitamente specifici strumenti di sviluppo software. Azure for Students fornisce l'accesso a tutti i prodotti di Azure espressamente destinati a sostenere l'istruzione o l'insegnamento, la ricerca non commerciale o le attività di progettazione, sviluppo, test e dimostrazione di applicazioni software per gli scopi indicati sopra. Gli studenti scaricano gli strumenti dall'hub di Education di Azure.

Azure for Students offre $ 100 di credito per 12 mesi. Include l'accesso a più di 25 servizi gratuiti, tra cui risorse di calcolo, rete, archiviazione e database. Eventuali addebiti sostenuti durante questo periodo vengono dedotti dal credito. È possibile verificare il credito rimanente nel [portale Azure Sponsorships](https://www.microsoftazuresponsorships.com/).

Dopo aver esaurito il credito disponibile o aver raggiunto il termine dei 12 mesi, la sottoscrizione di Azure viene disabilitata. La sottoscrizione di Azure for Students non è rinnovabile. Per continuare a usare Azure, è possibile eseguire l'aggiornamento a una sottoscrizione con pagamento in base al consumo nel [portale degli account Azure](https://account.azure.com/). Se si decide di non effettuare l'aggiornamento al termine dei 12 mesi o dopo aver esaurito il credito di $ 100, a seconda di quale delle due situazioni si verifichi per prima, tutti i prodotti distribuiti verranno rimossi e non sarà possibile accedervi. A partire dal termine della sottoscrizione gratuita, si hanno a disposizione 90 giorni per eseguire l'aggiornamento a una sottoscrizione con pagamento in base al consumo.

Per informazioni sulle condizioni per l'utilizzo di Azure for Students, vedere le [condizioni dell'offerta](https://azure.microsoft.com/offers/ms-azr-0170p/).

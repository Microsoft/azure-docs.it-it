# <a name="azure-for-students-starter"></a>Azure for Students Starter

Azure for Students Starter permette di iniziare a usare i prodotti di Azure che servono per sviluppare sul cloud. Non sono previsti costi. Questo vantaggio consente di accedere a un livello gratuito dei servizi seguenti:

- Servizio app di Azure
- Funzioni di Azure
- Hub di notifica di Azure
- Database di Azure per MySQL
- Application Insights
- Azure DevOps Server (in precedenza Visual Studio Team Services)

Azure for Students Starter è disponibile gratuitamente per gli studenti verificati e senza impegno o limite di tempo. Per informazioni dettagliate sulle condizioni per l'utilizzo, vedere l'[offerta Azure for Students Starter](https://azure.microsoft.com/offers/ms-azr-0144p/).

Questa offerta non è disponibile per l'uso in corsi MOOC (Massive Open Online Course) o in altri corsi professionali di formazione offerti da organizzazioni a fini di lucro.

Questa offerta è limitata a una sottoscrizione di Azure for Students Starter per ogni cliente idoneo. Non è trasferibile e non è cumulabile con altre offerte, se non diversamente autorizzato da Microsoft.

Per l'offerta Azure for Students Starter non è richiesta una carta di credito. Tuttavia, è possibile eseguire l'aggiornamento in qualsiasi momento a una sottoscrizione con pagamento in base al consumo per ottenere l'accesso a tutti i servizi di Azure, usando il [portale degli account Azure](https://account.azure.com/).

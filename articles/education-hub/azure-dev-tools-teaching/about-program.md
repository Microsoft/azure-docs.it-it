# <a name="what-is-azure-dev-tools-for-teaching"></a>Che cos'è Azure Dev Tools for Teaching?

Microsoft Azure Dev Tools for Teaching mette a disposizione gli strumenti di sviluppo, il software e i servizi professionali di Microsoft per docenti e studenti con sottoscrizioni a un costo contenuto. Gli studenti ricevono gratuitamente gli strumenti di sviluppo, ovvero tutto il necessario per creare app, giochi e siti Web, in modo che possano inseguire i propri sogni, creare innovazioni rivoluzionarie nella tecnologia o iniziare subito la loro carriera.

L'amministratore della sottoscrizione di Azure Dev Tools for Teaching potrà:

- Gestire e accedere al software tramite un portale online centralizzato che elimina la necessità di mantenere uno store Web o un sito interno.

- Offrire agli utenti accademici l'accesso a software, a materiali di formazione gratuiti e alla piattaforma cloud di Azure tramite lo stesso portale online.

## <a name="program-details"></a>Dettagli del programma

Il programma Azure Dev Tools for Teaching è destinato all'istruzione incentrata sulla metodologia STEM. Ogni piano di studi sui corsi di scienza, tecnologia, ingegneria o matematica è idoneo per l'uso di Azure Dev Tools for Teaching per aumentare l'efficacia dell'insegnamento per i professori e quella dell'apprendimento per gli studenti. 

La sottoscrizione di Microsoft Azure Dev Tools for Teaching offre l'accesso a determinati strumenti di sviluppo software. Questi strumenti sono disponibili gratuitamente per il download. Se un membro del corpo docente si è registrato in un corso approvato, è idoneo per installare il software di Azure Dev Tools for Teaching nel proprio computer personale per uso non commerciale.

Azure Dev Tools for Teaching è disponibile in più di 140 paesi/aree in cui Azure è disponibile in commercio. Lo store dell'hub di Education è disponibile nelle lingue seguenti: arabo, cinese semplificato, cinese tradizionale, coreano, danese, ebraico, francese, giapponese, inglese, italiano, olandese, portoghese, russo, spagnolo, svedese, tedesco e turco.

## <a name="student-benefits"></a>Vantaggi per gli studenti

Tutto il software è accessibile tramite un [hub di Education](https://azureforeducation.microsoft.com/devtools) (integrato nel [portale di Azure](https://portal.azure.com/)) che ospita tutti i prodotti e i servizi disponibili per gli utenti accademici. Gli studenti accedono al software semplicemente accedendo al portale di Azure con le stesse credenziali che usano per accedere alla loro sottoscrizione.

Gli studenti non devono iscriversi per ottenere una sottoscrizione di Azure per accedere ai vantaggi del software. Tuttavia, Microsoft offre $ 100 di credito Azure oltre all'accesso a servizi di Azure gratuiti per gli studenti, che possono iscriversi all'[offerta Azure for Students](azure-students-program.md) senza la necessità di una carta di credito.

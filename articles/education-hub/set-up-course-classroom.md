# <a name="quickstart-set-up-a-course-and-create-a-classroom"></a>Avvio rapido: Configurare un corso e creare una classe

Questo argomento di avvio rapido illustra come configurare un corso e una classe nell'hub di Education di Microsoft Azure, inclusi i dettagli sulle sottoscrizioni.

## <a name="prerequisites"></a>Prerequisiti

- Una sovvenzione accademica con un importo di credito approvato
- Una sottoscrizione nell'ambito della classe che accederà al credito Azure

### <a name="subscriptions"></a>Sottoscrizioni

A ogni studente viene assegnata una sottoscrizione associata a un limite monetario di credito allocato dal professore. Il termine *limite monetario* descrive l'importo in valuta di un contributo accademico. Un limite monetario 1.000 USD, ad esempio, fornisce al destinatario un credito Azure di 1.000 USD secondo le [tariffe WebDirect pubblicate](https://azure.microsoft.com/pricing/calculator/).

Il docente può scegliere di allocare un limite alle sottoscrizioni per evitare un uso non consentito e quindi impostare una data di scadenza. Di seguito alcuni esempi:

- **Importo fisso per corso**: ogni studente ottiene *x* dollari da gestire per l'intero trimestre o semestre.

- **Importo fisso per gruppo di progetto**: un team di progetto riceve*x* dollari per finanziare un progetto di gruppo.

A livello di sottoscrizione, è possibile aumentare o ridurre il limite e le modifiche verranno applicate con una latenza minima. Al termine del corso o del progetto, è possibile riallocare il limite inutilizzato ad altre sottoscrizioni prima della data di scadenza.

## <a name="create-a-course-and-classroom"></a>Creare un corso e una classe

Seguire questa procedura per creare un corso e una classe scegliendo tra due diversi metodi:

1. Selezionare la pagina **Corsi** nell'hub di Education di Azure per aprire lo strumento usato per creare e gestire i corsi. Verrà visualizzata una tabella che mostra tutti i corsi esistenti.

    :::image type="content" source="media/set-up-course-classroom/education-hub-courses.png" alt-text="Pagina Corsi dell'hub di Education di Azure" border="false":::

1. Selezionare l'icona **+ Aggiungi** nell'angolo superiore sinistro della tabella per avviare il flusso di lavoro di creazione.

    :::image type="content" source="media/set-up-course-classroom/courses-add.png" alt-text="Aggiungere un corso all'hub di Education di Azure" border="false":::

1. È possibile creare un registro del corso usando due metodi: caricando un registro o usando un codice di invito.
    - **Registro**: se si dispone già dei nomi e degli account di accesso di tutti gli studenti, è possibile popolare e caricare un file di registro. Per scaricare un file di esempio del file csv necessario per caricare il registro, selezionare il collegamento **Scarica file di esempio** nell'angolo in alto a destra.
    - **Codice di invito**: se si sceglie di usare un codice di invito, decidere quanti codici possono essere riscattati e definire la scadenza. Per riscattare il codice, si invierà agli studenti il collegamento seguente: https://aka.ms/joinedu.

      :::image type="content" source="media/set-up-course-classroom/enter-invitation-code.png" alt-text="Immettere il codice di invito nell'hub di Education di Azure" border="false":::

1. Selezionare **Crea classe** nell'angolo in basso a destra. Il completamento potrebbe richiedere alcuni istanti.

   :::image type="content" source="media/set-up-course-classroom/create-classroom.png" alt-text="Creare una classe nell'hub di Education di Azure" border="false":::

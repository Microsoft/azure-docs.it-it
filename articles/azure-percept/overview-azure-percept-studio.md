# <a name="azure-percept-studio-overview"></a>Panoramica di Azure Percept Studio

Azure Percept studio è il punto di lancio singolo per la creazione di soluzioni e modelli di intelligenza artificiale Edge. Azure Percept Studio ti permette di individuare e completare flussi di lavoro guidati che semplificano l'integrazione di hardware idoneo per intelligenza artificiale e servizi cloud di intelligenza artificiale e di Azure.

## <a name="features"></a>Funzionalità

In studio è possibile visualizzare i dispositivi perimetrali idonei per l'intelligenza artificiale come endpoint per la raccolta di dati di training iniziali e continui, nonché destinazioni di distribuzione per le iterazioni del modello. L'accesso ai dispositivi e ai dati di Training consente la creazione rapida di prototipi e il modello di intelligenza artificiale Edge iterativo per gli scenari di [visione](./tutorial-nocode-vision.md) e [sintesi vocale](./tutorial-no-code-speech.md) .

![Flusso di progetto dalla Home page in Azure Percept Studio.](./media/overview-azure-percept-studio/percept-studio-flow.png)

I flussi di lavoro in Azure Percept Studio integrano molti servizi di intelligenza artificiale e di gestione delle attività di Azure sottostanti, ad esempio l'hub Azure, Visione personalizzata, speech studio e i servizi di Azure ML, quindi è possibile usare questi servizi per creare una soluzione end-to-end, senza una conoscenza preesistente significativa. Se si ha già familiarità con questi servizi di Azure, è anche possibile connettersi e modificare le risorse esistenti al di fuori di Azure Percept Studio.

![Flusso del dispositivo in Azure Percept Studio.](./media/overview-azure-percept-studio/device-flow.png)

Indipendentemente dal fatto che si sia un programmatore di intelligenza artificiale e un modello di intelligenza artificiale più avanzato, lavorando a un prototipo o al passaggio a una soluzione di produzione, per l'intelligenza artificiale per i confini di sintesi vocale o visione artificiale, Azure Percept Studio consente di accedere ai flussi di lavoro che è possibile usare per ridurre i problemi relativi alla creazione di soluzioni di intelligenza artificiale

![Flusso di acquisizione di immagini in Azure Percept Studio.](./media/overview-azure-percept-studio/image-flow.png)

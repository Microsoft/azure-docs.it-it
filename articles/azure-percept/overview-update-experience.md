# <a name="azure-percept-dk-update-experience-overview"></a>Panoramica dell'esperienza di aggiornamento di Azure Percept DK

Con Azure Percept DK è possibile aggiornare il sistema operativo e il firmware del kit di sviluppo (OTA) o via USB. L'aggiornamento OTA è un modo semplice per gestire i dispositivi con l'aggiornamento del [dispositivo per](https://docs.microsoft.com/azure/iot-hub-device-update/) il servizio hub Internet. Gli aggiornamenti USB sono disponibili per gli utenti che non sono in grado di usare gli aggiornamenti OTA o quando è necessario un ripristino delle impostazioni predefinite del dispositivo. Per iniziare a usare gli aggiornamenti dei dispositivi di Azure Percept DK, vedere le guide seguenti:

- [Configurare l'hub Azure per la distribuzione di aggiornamenti in modalità wireless (OTA) in Azure Percept DK](./how-to-set-up-over-the-air-updates.md)
- [Aggiornare Azure Percept DK in modalità wireless (OTA)](./how-to-update-over-the-air.md)
- [Aggiornare Azure Percept DK su USB](./how-to-update-via-usb.md)

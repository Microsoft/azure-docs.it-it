# <a name="azure-percept-dk-8020-integration-overview"></a>Panoramica dell'integrazione di Azure Percept DK 80/20

Azure Percept DK e l'accessorio audio sono stati progettati per l'integrazione con il [sistema di ringhiera 80/20](https://8020.net/).

## <a name="8020-features"></a>funzionalità di 80/20

Ogni componente hardware viene compilato con le tacche e le sporgenze per adattarsi al tipo di estrusione 1010. Questa integrazione consente ai clienti e ai generatori di soluzioni di estendere più facilmente i concetti di prova negli ambienti di produzione.

# <a name="azure-percept-dk-8020-integration-overview"></a>Panoramica dell'integrazione di Azure Percept DK 80/20

Azure Percept DK e l'accessorio audio sono stati progettati per l'integrazione con il [sistema di compilazione in alluminio a 80/20 T-slot](https://8020.net/).

## <a name="8020-features"></a>funzionalità di 80/20

Azure Percept DK Carrier Board, Azure Percept Vision Device e Azure Percept audio accessori sono prodotti con connettori 80/20 1010 incorporati, che consentono configurazioni di montaggio infinite con le guide 80/20. Questa integrazione consente ai clienti e ai generatori di soluzioni di estendere più facilmente i concetti di prova negli ambienti di produzione.

# <a name="unity-showcase-application"></a>Applicazione di presentazione Unity

L'applicazione di presentazione Unity è un'applicazione di esempio basata su Unity per dimostrare le funzionalità di Rendering remoto di Azure. Può essere eseguita all'interno dell'editor di Unity e in Hololens 2.

![App di presentazione](./media/showcase-app.png "App di presentazione")

## <a name="source-repository"></a>Repository di origine

Il codice sorgente per questa applicazione, insieme alle istruzioni di compilazione/distribuzione, è reperibile nel [repository di esempi di Rendering remoto di Azure](https://github.com/Azure/azure-remote-rendering/tree/master/Unity/AzureRemoteRenderingShowcase).

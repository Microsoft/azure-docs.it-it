# <a name="azure-remote-rendering-asset-tool-arrt"></a>Strumento per gli asset di Rendering remoto di Azure

Lo strumento per gli asset di Rendering remoto di Azure è un'applicazione desktop open source sviluppata in C++/Qt.

![Strumento per gli asset di Rendering remoto di Azure](./media/azure-remote-rendering-asset-tool.png "Screenshot dello strumento per gli asset di Rendering remoto di Azure")

## <a name="features"></a>Funzionalità

L'applicazione può essere usata per:

* Caricare un modello 3D
* Controllare la conversione di modelli
* Creare e gestire una sessione di rendering remoto
* Caricare un modello 3D
* Visualizzare in anteprima il modello 3D
* Modificare i materiali

È anche possibile usarlo come esempio open source per apprendere come implementare un front-end per Azure Remote rendering C++ SDK, usando la libreria client di archiviazione di Azure per la gestione della conversione del modello 3D.

## <a name="source-repository"></a>Repository di origine

Il codice sorgente e la documentazione sono disponibili nel repository [GitHub ARRT](https://github.com/Azure/azure-remote-rendering-asset-tool).

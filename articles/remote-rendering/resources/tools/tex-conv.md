# <a name="texconv---texture-conversion-tool"></a>TexConv-strumento di conversione della trama

Lo strumento TexConv è uno strumento da riga di comando esterno per eseguire conversioni di trama in batch.
Lo strumento si trova in questo repository git: [repository GitHub per lo strumento TexConv](https://github.com/microsoft/DirectXTex/wiki/Texconv).

Per il rendering remoto, non è necessario convertire le trame, perché il passaggio di conversione può gestire i formati di file di immagine comuni e converte le trame nel formato di runtime appropriato. Tuttavia, questo strumento può essere utile per convertire le trame localmente con impostazioni personalizzate.

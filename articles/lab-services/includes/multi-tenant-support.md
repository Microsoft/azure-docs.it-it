Se si è un amministratore o un proprietario del Lab che fa parte di più tenant di Azure AD, è possibile spostarsi tra i tenant dal portale di Azure Lab Services (l'angolo superiore destro). 

![Seleziona un tenant](../media/multi-tenant-support/picker.png)

Dopo aver selezionato un tenant, selezionare un account Lab all'interno del tenant Azure Active Directory (Azure AD) nell'angolo superiore sinistro.

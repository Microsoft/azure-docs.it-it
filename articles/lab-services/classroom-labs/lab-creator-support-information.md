# <a name="view-support-information-lab-creator-in-azure-lab-services"></a>Visualizzare le informazioni di supporto (Lab Creator in Azure Lab Services)
Questo articolo illustra come l'autore del Lab può visualizzare le informazioni di supporto seguenti:

- URL
- Posta elettronica
- Telefono
- Istruzioni aggiuntive

È possibile usare queste informazioni per ottenere assistenza quando si verificano problemi tecnici durante la creazione di un Lab in un account Lab.

 
## <a name="view-support-information"></a>Visualizzare le informazioni di supporto
1. Accedere al [portale di Azure Lab Services](https://labs.azure.com).
2. Selezionare punto interrogativo (**?**) nell'angolo superiore destro della pagina. 
3. Verificare che siano visualizzati i collegamenti al **sito Web del supporto tecnico**, al **supporto di posta elettronica**e al **numero di telefono del supporto**.

    ![Visualizzare le informazioni di supporto](../media/lab-creator-support-information/support-information.png)

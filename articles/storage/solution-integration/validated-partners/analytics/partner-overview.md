# <a name="azure-storage-analytics-and-big-data-partners"></a>Analisi archiviazione di Azure e partner Big Data

In questo articolo vengono evidenziate le società partner Microsoft integrate con Azure Data Lake Storage. Queste soluzioni partner coprono i carichi di lavoro, ad esempio i carichi di lavoro data warehouse moderni, l'analisi avanzata e l'analisi in tempo reale. Questi partner sfruttano lo [spazio dei nomi gerarchico](../../../blobs/data-lake-storage-namespace.md) in archiviazione di Azure per ottimizzare la propria soluzione ed eseguirla in modo efficiente in Azure.

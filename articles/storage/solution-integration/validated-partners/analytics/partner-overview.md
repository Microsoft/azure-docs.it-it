# <a name="azure-storage-analytics-and-big-data-partners"></a>Partner Analisi archiviazione di Azure e Big Data

In questo articolo vengono evidenziate le società partner Microsoft integrate con Azure Data Lake Storage. Coprono i carichi di lavoro, inclusi data warehouse moderni, analisi avanzate, analisi in tempo reale e così via. Questi partner sfruttano lo spazio dei nomi gerarchico in archiviazione di Azure per ottimizzare la propria soluzione ed eseguirla in modo efficiente in Azure.

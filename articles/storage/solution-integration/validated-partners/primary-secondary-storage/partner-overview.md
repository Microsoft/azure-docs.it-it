# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>Partner di archiviazione di Azure per l'archiviazione primaria e secondaria

In questo articolo vengono evidenziate le società partner Microsoft che forniscono una soluzione NAS (Network Attached Storage) o SAN (Storage Area Network). La soluzione può essere locale, in Azure o una soluzione ibrida che usa archiviazione di Azure come livello economico. Queste soluzioni consentono ai clienti di usare la stessa soluzione in qualsiasi ambiente.

# <a name="azcopy-load"></a>azcopy load

Sottocomandi correlati al trasferimento di dati in formati specifici

## <a name="synopsis"></a>Riepilogo

Sottocomandi correlati al trasferimento di dati in formati specifici, ad esempio il formato CLFS (Microsoft havee cloud FileSystem).

## <a name="examples"></a>Esempi

Caricare un'intera directory in un contenitore con una firma di accesso condiviso nel formato CLFS:

```azcopy
azcopy load clfs "/path/to/dir" "https://[account].blob.core.windows.net/[container]?[SAS]" --state-path="/path/to/state/path"
```

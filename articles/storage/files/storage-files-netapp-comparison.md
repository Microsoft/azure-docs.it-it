# <a name="azure-files-and-azure-netapp-files-comparison"></a>File di Azure e Azure NetApp Files confronto

Questo articolo offre un confronto tra File di Azure e Azure NetApp Files. 

La maggior parte dei carichi di lavoro che richiedono l'archiviazione file cloud funziona correttamente File di Azure o Azure NetApp Files. Per determinare la soluzione migliore per il carico di lavoro, esaminare le informazioni fornite in questo articolo. Per altre informazioni, vedere la [documentazione File di Azure](https://docs.microsoft.com/azure/storage/files/) e [Azure NetApp Files.](https://docs.microsoft.com/azure/azure-netapp-files/)

# <a name="azure-files-and-azure-netapp-files-comparison"></a>Confronto tra File di Azure e Azure NetApp Files

Questo articolo fornisce un confronto tra File di Azure e Azure NetApp Files. 

La maggior parte dei carichi di lavoro che richiedono l'archiviazione file Cloud funziona correttamente in File di Azure o Azure NetApp Files. Per determinare la soluzione più adatta al carico di lavoro, esaminare le informazioni fornite in questo articolo. Per ulteriori informazioni, vedere la documentazione relativa a [file di Azure](https://docs.microsoft.com/azure/storage/files/) e [Azure NetApp files](https://docs.microsoft.com/azure/azure-netapp-files/) .

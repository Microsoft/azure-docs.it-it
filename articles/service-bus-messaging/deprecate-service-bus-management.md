# <a name="deprecation-of-azure-service-manager-support-for-azure-service-bus-relay-and-event-hubs"></a>Deprecazione del supporto Service Manager azure per bus di servizio di Azure, inoltro e Hub eventi

Resource Manager, lo stack dell'infrastruttura cloud di nuova generazione, sostituisce completamente il modello di gestione dei servizi di Azure "classico" (modello di distribuzione classica). Di conseguenza, le API REST del modello di distribuzione classica e il supporto per bus di servizio, inoltro e hub eventi verranno ritirati il 1° novembre 2021. Questa deprecazione è stata annunciata per la prima volta in un annuncio della [Microsoft Tech Community,](https://techcommunity.microsoft.com/t5/Service-Bus-blog/Deprecating-Service-Management-support-for-Azure-Service-Bus/ba-p/370909)ma di recente è stato deciso di estendere il periodo di deprecazione di altri due anni dal momento dell'annuncio originale. Per una facile identificazione, queste API sono `management.core.windows.net` presenti nell'URI. Fare riferimento alla tabella seguente per un elenco delle API deprecate e della relativa Azure Resource Manager API che è ora consigliabile usare.

Per continuare a usare bus di servizio, inoltro e Hub eventi, passare a Resource Manager entro il 31 ottobre 2021. Si consiglia a tutti i clienti che usano ancora API vecchie di passare a breve per sfruttare i vantaggi aggiuntivi di Resource Manager, tra cui raggruppamento delle risorse, tag, un processo di distribuzione e gestione semplificato e un controllo degli accessi con granularità fine usando il controllo degli accessi in base al ruolo di Azure.

Per altre informazioni su Azure Resource Manager confronto tra azure Service Manager, vedere il [blog di TechNet.](https://blogs.technet.microsoft.com/meamcs/2016/12/22/difference-between-azure-service-manager-and-azure-resource-manager/)

Per altre informazioni sulle API Service Manager e Resource Manager per bus di servizio di Azure, inoltro e Hub eventi, vedere la documentazione dell'API REST:

- [Bus di servizio di Azure](/rest/api/servicebus/)
- [Hub eventi di Azure](/rest/api/eventhub/)
- [Inoltro di Azure](/rest/api/relay/)

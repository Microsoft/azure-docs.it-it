# <a name="azure-policy-regulatory-compliance-controls-for-azure-service-bus-messaging"></a>Controlli di conformità alle normative di Criteri di Azure per la messaggistica del bus di servizio di Azure

La [conformità alle normative di Criteri di Azure](../governance/policy/concepts/regulatory-compliance.md) offre definizioni di iniziative create e gestite da Microsoft, note come _definizioni predefinite_, per i **domini di conformità** e i **controlli di sicurezza** correlati a diversi standard di conformità. Questa pagina elenca i **domini di conformità** e i **controlli di sicurezza** per la messaggistica del bus di servizio di Azure. È possibile assegnare singolarmente le definizioni predefinite per un **controllo di sicurezza** in modo da rendere le risorse di Azure conformi allo specifico standard.

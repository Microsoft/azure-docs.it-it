# <a name="legacy-devices-on-the-azure-certified-device-catalog"></a>Dispositivi legacy nel catalogo dei dispositivi Azure Certified

Probabilmente si è notato nel catalogo dei dispositivi Azure Certified che alcuni dispositivi non hanno il contorno blu o l'etichetta "dispositivo Azure Certified". Questi dispositivi (denominati "dispositivi legacy") erano precedentemente certificati nel programma legacy.

## <a name="certified-for-azure-iot-program"></a>Programma certificate for Azure Internet

Prima del lancio del programma Azure Certified Device, i partner hardware avrebbero potuto certificare i prodotti in precedenza nel programma di Azure. Il programma Azure Certified Device Certification consente di concentrare la propria missione per fornire le promesse dei clienti anziché le funzionalità tecniche del dispositivo.

I dispositivi che sono stati certificati come un "dispositivo certificato dell'hub Internet" vengono visualizzati nel catalogo di dispositivi certificati di Azure come "dispositivo legacy". Questa etichetta indica che i dispositivi sono stati precedentemente qualificati tramite il programma attualmente ritirato, ma che non sono stati certificati tramite il programma Azure Certified Device aggiornato. Questi dispositivi sono chiaramente indicati nel catalogo in base alla mancanza di struttura blu ed è possibile trovare il filtro "dispositivi certificati hub Internet (legacy)".

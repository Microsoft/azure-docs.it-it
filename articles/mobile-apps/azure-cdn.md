# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Content delivery network on Azure with Azure Content Delivery Network
[La rete](https://azure.microsoft.com/services/cdn/) per la distribuzione di contenuti di Azure offre agli sviluppatori una soluzione globale per distribuire rapidamente contenuti ad alta larghezza di banda come video, immagini o file statici agli utenti memorizzandone i contenuti nella cache in centinaia di punti di presenza in tutto il mondo. La rete per la distribuzione di contenuti può anche accelerare il contenuto dinamico, che non può essere memorizzato nella cache, per ridurre i tempi di caricamento, risparmiare larghezza di banda e aumentare la velocità di risposta. Si ottengono questi vantaggi sia che si sviluppino o se si gestiscono siti Web o applicazioni per dispositivi mobili o si codichino e si distribuiscano contenuti multimediali in streaming, software di gioco, aggiornamenti del firmware o endpoint IoT.

## <a name="azure-content-delivery-network-features"></a>Funzionalità della rete per la distribuzione di contenuti di AzureAzure Content Delivery Network features
  - Lavora senza problemi con i servizi di Azure come applicazioni web, servizi multimediali, archiviazione e servizi cloud.
  - Gestisci facilmente i picchi di traffico e i carichi pesanti.
  - Integrati con Akamai e Verizon per creare e personalizzare la strategia di distribuzione dei contenuti.
  - Utilizzare l'accelerazione dinamica del sito.
  - Controllare il comportamento di memorizzazione nella cache della rete per la distribuzione di contenuti con le regole di memorizzazione nella cache.
  - Ottieni il supporto del dominio personalizzato HTTPS.
  - Visualizzare i log di diagnostica di Azure per visualizzare l'analisi di base ed esportarli in Archiviazione BLOB di Azure, Hub eventi di Azure o Monitoraggio di Azure.View Azure diagnostics logs to see core analytics and export them to Azure Blob storage, Azure Event Hubs, or Azure Monitor.
  - Migliorare le prestazioni comprimendo i file.
  - Utilizzare il geofiltering per limitare il contenuto della rete per la distribuzione di contenuti in base al paese o all'area geografica.

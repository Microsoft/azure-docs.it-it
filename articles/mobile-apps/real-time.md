# <a name="build-real-time-applications-with-azure-signalr-service"></a>Creare applicazioni in tempo reale con il servizio SignalR di AzureBuild real-time applications with Azure SignalR Service

Con [il servizio SignalR](https://azure.microsoft.com/services/signalr-service/)di Azure è possibile aggiungere facilmente funzionalità in tempo reale alle applicazioni. Questa funzionalità in tempo reale consente al servizio di eseguire il push degli aggiornamenti dei contenuti nelle applicazioni mobili connesse. Di conseguenza, le app vengono aggiornate senza la necessità di eseguire il polling del server o di inviare nuove richieste HTTP per gli aggiornamenti.

## <a name="azure-signalr-service-features"></a>Funzionalità del servizio SignalR di AzureAzure SignalR Service features
- Dispone di un ampio supporto client per un'ampia gamma di client, ad esempio browser Web e mobili, applicazioni desktop, processi server, dispositivi IoT e console di gioco.
- Consente a più istanze di lavorare insieme per scalare a milioni di connessioni di applicazioni mobili. Supporta inoltre più aree globali per scopi di partizionamento di partizione, disponibilità elevata o ripristino di emergenza.
- Offre API avanzate per modelli di messaggistica diversi per inviare messaggi a una determinata connessione, a tutte le connessioni o a un sottoinsieme di connessioni che appartengono a un utente specifico o che sono state inserite in un gruppo arbitrario.

# <a name="mapping-of-azure-security-benchmark-v2-and-cis-microsoft-azure-foundations-benchmark"></a>Mapping del benchmark di sicurezza di Azure V2 e del benchmark delle fondazioni di Microsoft Azure CIS

Il benchmark della Microsoft Azure Foundation di CIS è il materiale sussidiario di sicurezza fornito da Center for Internet Security per stabilire una configurazione di base sicura per Azure. L'ambito del benchmark consiste nel definire il livello di sicurezza di base per l'adozione del cloud di Azure. Questo benchmark è in linea con il [benchmark di sicurezza di Azure V2](overview.md).

Viene fornito un mapping tra il Microsoft Azure di base del [benchmark di CIS versione v 1.3.0](https://www.cisecurity.org/benchmark/azure/) e il [benchmark di sicurezza di Azure V2](overview.md). Ciò consentirà di comprendere come sono allineate le raccomandazioni per entrambi i benchmark. È possibile scaricare il mapping completo [qui](https://github.com/MicrosoftDocs/SecurityBenchmarks/blob/master/Azure%20Security%20Benchmark/2.0/asb_v2_to_cis_microsoft_azure_foundations_benchmark_v1.3.0.xlsx).

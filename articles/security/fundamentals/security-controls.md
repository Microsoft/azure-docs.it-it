# <a name="index-of-built-in-security-controls-articles-for-azure-services"></a>Indice degli articoli "Controlli di sicurezza incorporati" per i servizi di AzureIndex of "Built-in security controls" articles for Azure services

Questo indice fornisce collegamenti agli articoli "Controlli di sicurezza incorporati". Un controllo di sicurezza è una qualità o funzionalità di un servizio di Azure che contribuisce alla capacità del servizio di prevenire, rilevare e rispondere alle vulnerabilità della sicurezza.

Gli articoli di controllo di sicurezza incorporati sono disponibili per i servizi seguenti:Built-in security control articles are available for the following services:

- [Gestione API](../../api-management/api-management-security-controls.md)
- [Servizio app di Azure](../../app-service/app-service-security-controls.md)
- [Azure Resource Manager](../../azure-resource-manager/management/azure-resource-manager-security-controls.md)
- [Azure Backup](../../backup/backup-security-controls.md)
- [Hub eventi di Azure](../../event-hubs/event-hubs-security-controls.md)
- [ExpressRoute di Azure](../../expressroute/expressroute-security-controls.md)
- [Servizio di bilanciamento del carico di AzureAzure Load Balancer](../../load-balancer/load-balancer-security-controls.md)
- [Messaggistica del bus di servizio di AzureAzure Service Bus Messaging](../../service-bus-messaging/service-bus-messaging-security-controls.md)
- [Inoltro del bus di servizio di Azure](../../service-bus-relay/service-bus-relay-security-controls.md)
- [Azure Service Fabric](../../service-fabric/service-fabric-security-controls.md)
- [Database SQL di Azure](../../sql-database/sql-database-security-controls.md)
- [Set di scalabilità delle macchine virtuali di Azure](../../virtual-machine-scale-sets/virtual-machine-scale-sets-security-controls.md)
- [Macchine virtuali Linux](../../virtual-machines/linux/virtual-machines-linux-security-controls.md)

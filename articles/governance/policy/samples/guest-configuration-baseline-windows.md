|Controllo: arresto del sistema immediato se non è possibile registrare i controlli di sicurezza<br /><sub>(CCE-35907-5)</sub> |**Descrizione**: questa impostazione dei criteri determina se il sistema viene arrestato se non è in grado di registrare gli eventi di sicurezza. È necessario che i criteri di valutazione del sistema (TCSEC) attendibili-C2 e la certificazione dei criteri comuni impediscano la verifica degli eventi controllabili se il sistema di controllo non è in grado di registrarli. Microsoft ha scelto di soddisfare questo requisito arrestando il sistema e visualizzando un messaggio di arresto nel caso in cui si verifichi un errore nel sistema di controllo. Quando questa impostazione dei criteri è abilitata, il sistema viene arrestato se non è possibile registrare un controllo di sicurezza per qualsiasi motivo. Se l'impostazione controlla: Arresta sistema immediatamente se non è possibile registrare i controlli di sicurezza è abilitata, possono verificarsi errori di sistema non pianificati. Il carico amministrativo può essere significativo, soprattutto se si configura anche il metodo di conservazione per il registro di protezione per non sovrascrivere eventi (Cancella registro manualmente).

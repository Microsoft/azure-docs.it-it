# <a name="overview-of-the-dod-impact-level-5-blueprint-sample"></a>Panoramica dell'esempio di progetto DoD Impact Level 5

L'esempio di progetto Department of Defense Impact Level 5 (DoD IL5) fornisce misure di tutela per la governance tramite [Criteri di Azure](../../../policy/overview.md) che consentono di valutare specifici controlli DoD Impact Level 5. Il progetto aiuta i clienti a distribuire un set di base di criteri per qualsiasi architettura distribuita in Azure che deve implementare i controlli DoD Impact Level 5. Per informazioni più aggiornate su quali cloud e servizi di Azure soddisfano l'autorizzazione DoD Impact Level 5, vedere [Servizi di Azure in base all'ambito di controllo FedRAMP e DoD CC SRG](../../../../azure-government/compliance/azure-services-in-fedramp-auditscope.md).

> [!NOTE]
> Questo esempio di progetto è disponibile in Azure per enti pubblici.

## <a name="control-mapping"></a>Mapping dei controlli

La sezione sul mapping dei controlli fornisce informazioni sui criteri inclusi in questo progetto e su come soddisfano i vari controlli DoD Impact Level 5. Le risorse assegnate a un'architettura vengono valutate da Criteri di Azure per rilevare la mancata conformità ai criteri assegnati. Per altre informazioni, vedere [Criteri di Azure](../../../policy/overview.md).

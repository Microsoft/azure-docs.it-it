# <a name="overview-of-the-media-blueprint-sample"></a>Panoramica dell'esempio di progetto Media

L'esempio di progetto Media fornisce una serie di misure di tutela per la governance tramite [Criteri di Azure](../../../policy/overview.md) che contribuiscono ad acquisire l'attestazione [Media](https://www.hhs.gov/hipaa/for-professionals/security/laws-regulations/index.html). 

## <a name="blueprint-sample"></a>Esempio di progetto

L'esempio di progetto consente ai clienti di distribuire un set di base di criteri per qualsiasi architettura distribuita in Azure che richiede l'accreditamento o la conformità al framework Media. La sezione sul [mapping dei controlli](./control-mapping.md) fornisce informazioni sui criteri inclusi in questa iniziativa e su come soddisfano i vari controlli definiti dal framework Media. Le risorse assegnate a un'architettura vengono valutate da Criteri di Azure per rilevare la conformità ai criteri assegnati.

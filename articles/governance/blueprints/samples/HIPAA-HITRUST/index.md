# <a name="overview-of-the-hipaa-hitrust-blueprint-sample"></a>Panoramica dell'esempio di progetto per i controlli HIPAA HITRUST

L'esempio di progetto HIPAA HITRUST fornisce una serie di misure di tutela per la governance tramite [Criteri di Azure](../../../policy/overview.md) che contribuiscono ad acquisire l'attestazione [HIPAA HITRUST](https://www.hhs.gov/hipaa/for-professionals/security/laws-regulations/index.html). 

## <a name="blueprint-sample"></a>Esempio di progetto

L'esempio di progetto consente ai clienti di distribuire un set di base di criteri per qualsiasi architettura distribuita in Azure che richiede l'accreditamento o la conformità al framework HIPAA HITRUST. La sezione sul [mapping dei controlli](./control-mapping.md) fornisce informazioni sui criteri inclusi in questa iniziativa e su come tali criteri contribuiscono a soddisfare i vari controlli definiti dal framework HITRUST HIPAA. Le risorse assegnate a un'architettura vengono valutate da Criteri di Azure per rilevare la conformità ai criteri assegnati.

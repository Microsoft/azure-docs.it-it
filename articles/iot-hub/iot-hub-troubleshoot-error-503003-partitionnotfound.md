# <a name="503003-partitionnotfound"></a>503003 PartitionNotFound

Questo articolo descrive le cause e le soluzioni per gli errori di **503003 PartitionNotFound** .

## <a name="symptoms"></a>Sintomi

Le richieste all'hub Internet non riescono con l'errore **503003 PartitionNotFound**.

## <a name="cause"></a>Causa

Questo errore è interno all'hub Internet e probabilmente è temporaneo. Vedere l' [errore interno del server dell'hub](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause)Internet.

## <a name="solution"></a>Soluzione

Vedere le [soluzioni per gli errori interni del server dell'hub](iot-hub-troubleshoot-error-500xxx-internal-errors.md#solution)Internet.

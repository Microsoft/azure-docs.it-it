# <a name="streaming-catalog-views-transact-sql"></a>Viste del catalogo di streaming (Transact-SQL)

Questa sezione contiene le viste e le funzioni del catalogo disponibili correlate a Streaming T-SQL.
  
## <a name="in-this-section"></a>Contenuto della sezione  
  
|Visualizzazione|Descrizione|  
|:---|:---|
|[sys.external_streams](sys-external-streams.md) |Restituisce una riga per ogni oggetto flusso esterno creato nell'ambito del database.|
|[sys.external_streaming_jobs](sys-external-streaming-jobs.md) |Restituisce una riga per ogni processo di streaming esterno creato nell'ambito del database.|
|[sys.external_job_streams](sys-external-job-streams.md)|Restituisce una riga per ogni oggetto flusso esterno di input o di output mappato a un processo di streaming esterno.|

# <a name="azure-policy-regulatory-compliance-controls-for-azure-app-configuration"></a>Criteri di Azure controlli di conformità alle normative per Configurazione app di Azure

La [conformità alle normative di Criteri di Azure](../governance/policy/concepts/regulatory-compliance.md) fornisce definizioni di iniziative gestite e create da Microsoft, note come _definizioni predefinite_, per i **domini di conformità** e i **controlli di sicurezza** correlati a diversi standard di conformità. Questa pagina elenca i domini **di conformità e** i controlli di **sicurezza** per Configurazione app di Azure. È possibile assegnare singolarmente i criteri predefiniti per un **controllo di sicurezza** in modo da rendere le risorse di Azure conformi allo specifico standard.

# <a name="azure-cdn-ddos-protection"></a>Protezione DDoS CDN di AzureAzure CDN DDoS Protection

Una rete per la distribuzione di contenuti fornisce protezione DDoS in base alla progettazione. Oltre alla capacità globale di assorbire gli attacchi volumetrici, la rete CDN di Azure ha una protezione DDoS aggiuntiva, come descritto di seguito, senza costi aggiuntivi.

## <a name="azure-cdn-from-microsoft"></a>Rete CDN di Azure di Microsoft

La rete CDN di Azure di Microsoft è protetta da [DDoS di base](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview)di Azure. È integrato nella rete CDN di Azure dalla piattaforma Microsoft per impostazione predefinita e senza costi aggiuntivi. L'intera portata e capacità della rete CDN di Azure dalla rete distribuita a livello globale di Microsoft fornisce difesa dagli attacchi comuni a livello di rete tramite il monitoraggio del traffico sempre attivo e la mitigazione in tempo reale. La protezione DDoS di base difende anche i più comuni attacchi di query DNS di livello 7 e di livello 3 e 4 che colpiscono gli endpoint della rete CDN. Questo servizio ha anche una comprovata esperienza nella protezione dei servizi aziendali e consumer di Microsoft da attacchi su larga scala.

## <a name="azure-cdn-from-verizon"></a>Rete CDN di Azure di Verizon

La rete CDN di Azure di Verizon è protetta dalla piattaforma di mitigazione DDoS proprietaria di Verzion. È integrato nella rete CDN di Azure di Verizon per impostazione predefinita e senza costi aggiuntivi. Fornisce protezione di base contro i più comuni attacchi di query DNS di livello 7 e di cui ai frequenti che si verificano e gli attacchi volumetrici di livello 3 e 4 destinati agli endpoint della rete CDN.

## <a name="azure-cdn-from-akamai"></a>Rete CDN di Azure di Akamai

La rete CDN di Azure di Akamai è protetta dalla piattaforma di mitigazione DDoS proprietaria di Akamai. È integrato nella rete CDN di Azure di Akamai per impostazione predefinita e senza costi aggiuntivi. Fornisce protezione di base contro i più comuni attacchi di query DNS di livello 7 e di cui ai frequenti che si verificano e gli attacchi volumetrici di livello 3 e 4 destinati agli endpoint della rete CDN.

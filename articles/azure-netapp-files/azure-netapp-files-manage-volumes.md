# <a name="manage-volumes-by-using-azure-netapp-files"></a>Gestione dei volumi tramite Azure NetApp Files
È possibile gestire un volume visualizzandolo, aggiornandolo o eliminandolo. 

Se si desidera eliminare un volume, è necessario eliminare innanzitutto tutti gli oggetti secondari di un volume. 

1.  Scegliere il pannello **Volumi** nel pannello Manage Capacity Pools (Gestisci pool di capacità). 
2.  Usare la vista **Browse Volumes** (Sfoglia volumi) per visualizzare, aggiornare o eliminare i volumi esistenti.

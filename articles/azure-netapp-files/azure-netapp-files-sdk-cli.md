## <a name="azure-resource-manager-templates"></a>Modelli di Gestione risorse di Azure  

Azure Resource Manager (ARM) consente di effettuare il provisioning delle applicazioni usando un modello dichiarativo. In un unico modello, è possibile distribuire più servizi con le relative dipendenze. Si usa lo stesso modello per distribuire più volte l'applicazione durante ogni fase del ciclo di vita dell'applicazione.   

Vedere [modelli ARM disponibili per Azure NetApp files](https://azure.microsoft.com/resources/templates/?term=anf).

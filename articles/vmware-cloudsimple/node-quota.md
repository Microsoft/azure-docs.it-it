# <a name="cloudsimple-node-quota-limits"></a>Limiti di quota dei nodi CloudSimple

Quattro nodi è la quantità predefinita disponibile per l'acquisto quando l'abbonamento è abilitato per il servizio CloudSimple.  È possibile acquistare qualsiasi [tipo di nodo](cloudsimple-node.md) dal portale di Azure.You can purchase any node type from the Azure portal.  Per creare un cloud privato sono necessari almeno tre nodi dello stesso SKU.  Se hai acquistato i nodi, potresti visualizzare un errore quando tenti di acquistare nodi aggiuntivi.

## <a name="quota-increase"></a>Aumento di quota

È possibile aumentare la quota del nodo inviando una richiesta di supporto. Il team delle operazioni del servizio valuterà la richiesta e collaborerà con l'utente per aumentare la quota del nodo.  Selezionare le seguenti opzioni quando si apre un nuovo ticket:

* Tipo di problema: **Tecnico**
* Abbonamento: **l'ID dell'abbonamento**
* Tipo di servizio: **VMware Solution by CloudSimple**
* Tipo di problema: **quota di nodi dedicati**
* Sottotipo di problema: **aumentare la quota dei nodi dedicati**
* Oggetto: **Aumento delle quote**

Nei dettagli del ticket di supporto, specificare il numero richiesto di nodi e SKU di nodo.

* SKU del nodo
* Numero di nodi aggiuntivi per i quali si richiede l'aumento della quota

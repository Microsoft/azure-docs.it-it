# <a name="what-is-azure-vmware-solution-by-cloudsimple"></a>Che cos'è la soluzione Azure VMware di CloudSimpleWhat is Azure VMware Solution by CloudSimple

**Azure VMware Solution by CloudSimple** è un servizio completamente gestito che consente di eseguire la piattaforma VMware in Azure.Azure VMware Solution by CloudSimple is a fully managed service that empowers you to run the VMware platform in Azure. Questa soluzione include vSphere, vCenter, vSAN, NSX-T e gli strumenti corrispondenti. L'ambiente VMware viene eseguito in modo nativo nell'infrastruttura bare metal di Azure nelle posizioni cloud di Azure.Your VMware environment runs natively on Azure bare metal infrastructure in Azure cloud locations. Il servizio include tutte le funzionalità necessarie per utilizzare le piattaforme VMware in modo efficiente e sicuro.

![Panoramica di VMware Solution on Azure by CloudSimple](media/azure-vmware-solution-by-cloudsimple.png)

## <a name="features"></a>Funzionalità

* Provisioning self-service su richiesta di ambienti cloud VMware. Possibilità di aggiungere e rimuovere capacità su richiesta.
* Distribuzione, aggiornamento, backup del piano di gestione, monitoraggio dell'integrità/capacità, avvisi, risoluzione dei problemi e correzione della piattaforma VMware.
* Servizi di rete sottoposti a requisiti necessari per abilitare VMware, inclusi i servizi L2/L3 e la gestione delle regole firewall.
* Servizi di rete di tipo Edge, tra cui VPN, IP pubblico e gateway Internet. Questi servizi vengono eseguiti in Azure e supportano la sicurezza e la protezione DDoS di Azure.These services run on Azure and carry the security and DDoS protection of Azure.
* Prenotazione della capacità per ridurre i costi.
* Connettività ad alta velocità e a bassa latenza ad Azure e in locale.
* Architetture di soluzioni per i clienti di utilizzare i servizi di Azure in modo integrato e sfruttare questa architettura unica "VMware cloud in un cloud pubblico". I servizi di Azure includono Azure AD, archiviazione, gateway applicazione e altri.
* Infrastruttura completamente dedicata all'utente e fisicamente isolata dall'infrastruttura di altri clienti.
* Funzionalità di gestione quali la gestione delle attività, l'utilizzo, la fatturazione/misurazione e la gestione degli utenti.
* Assistenza clienti 24x7.

* **Agilità dell'infrastruttura**. Non è più necessario prevedere tutte le esigenze di capacità e quindi con una capacità sprecata o una carenza di infrastrutture. CloudSimple viene fornito come servizio cloud ed è possibile aggiungere o ridurre la capacità in qualsiasi momento
* **Sicurezza**. L'accesso all'ambiente CloudSimple tramite Azure offre protezione DDoS incorporata e monitoraggio della sicurezza.
* **Costi inferiori**. La piattaforma CloudSimple è altamente progettata e fornisce elevati livelli di automazione, efficienza operativa ed economie di scala. I team di sviluppo possono sfruttare le piattaforme pubbliche e private in modo integrato e coerente.
* **Monitoraggio, risoluzione dei problemi e supporto dell'infrastruttura**. CloudSimple gestisce l'infrastruttura sottostante come servizio. L'hardware in errore viene sostituito automaticamente. * **Ritiro o migrazione del data center**. Ottenere capacità aggiuntiva quando si raggiungono i limiti del data center esistente o si aggiorna l'hardware. È facile aggiungere la capacità necessaria nel cloud ed eliminare i problemi della gestione degli aggiornamenti hardware. Riduci i rischi e i costi delle migrazioni cloud rispetto alle conversioni o alla riarchitettura che richiedono molto tempo. Utilizza strumenti e competenze VMware familiari per accelerare le migrazioni cloud. Nel cloud usare i servizi di Azure per modernizzare le applicazioni in base alle proprie esigenze.
* **Espandere su richiesta**. Espandere nel cloud per soddisfare esigenze impreviste, ad esempio nuovi ambienti di sviluppo o picchi di capacità stagionali. È possibile creare facilmente una nuova capacità su richiesta e mantenerla solo per tutto il tempo necessario. Riduci l'investimento iniziale, accelera la velocità del provisioning e complessità con la stessa architettura e i criteri sia in locale che nel cloud.
* **Ripristino di emergenza e desktop virtuali nel cloud**di Azure. Stabilisci l'accesso remoto a dati, app e desktop nel cloud di Azure.Establish remote access to data, apps, and desktops in the Azure cloud. Con le connessioni ad alta larghezza di banda, è possibile caricare e scaricare rapidamente i dati per il ripristino da incidenti. Le reti a bassa latenza offrono tempi di risposta rapidi che gli utenti si aspettano da un'app desktop. La facilità di ripristino e replica riduce notevolmente lo sforzo e il rischio di creare e gestire implementazioni DR e VDI.

Eseguire Oracle, Microsoft SQL Server, sistemi middleware e database no-SQL ad alte prestazioni. Sperimenta il cloud come tuo data center con connessioni di rete ad alta velocità da 25 Gbps che ti consentono di eseguire app ibride che si estendono in locale, VMware in Azure e carichi di lavoro privati di Azure senza compromettere le prestazioni.

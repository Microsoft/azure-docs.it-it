# <a name="azure-policy-regulatory-compliance-controls-for-azure-api-for-fhir"></a>Criteri di Azure controlli di conformità alle normative per API di Azure per FHIR

La [conformità alle normative di Criteri di Azure](../../governance/policy/concepts/regulatory-compliance.md) fornisce definizioni di iniziative gestite e create da Microsoft, note come _definizioni predefinite_, per i **domini di conformità** e i **controlli di sicurezza** correlati a diversi standard di conformità. Questa pagina elenca i domini **di conformità e** i controlli di **sicurezza** per API di Azure per FHIR. È possibile assegnare singolarmente i criteri predefiniti per un **controllo di sicurezza** in modo da rendere le risorse di Azure conformi allo specifico standard.

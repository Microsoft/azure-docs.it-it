# <a name="migration-solutions-for-azure-vmware-solution-virtual-machines-vms"></a>Soluzioni di migrazione per macchine virtuali della soluzione VMware di Azure (VM)

Uno dei casi d'uso più comuni per l'uso della soluzione VMware di Azure è data center l'evacuazione.  Consente di continuare a massimizzare gli investimenti VMware, perché la soluzione VMware di Azure sarà sempre aggiornata. Inoltre, è possibile migliorare i carichi di lavoro con l'intera gamma di servizi nativi di Azure.  Un passaggio iniziale della chiave in questo processo è la migrazione dell'ambiente basato su VMware legacy alla soluzione VMware di Azure.

I nostri partner di migrazione hanno soluzioni di migrazione leader nel settore negli ambienti basati su VMware. I clienti di tutto il mondo hanno usato queste soluzioni per le migrazioni in Azure e nella soluzione VMware di Azure.

Non è necessario usare VMware HCX come strumento di migrazione. Ciò significa che è anche possibile eseguire la migrazione dei carichi di lavoro fisici nella soluzione VMware di Azure. Inoltre, le migrazioni all'ambiente della soluzione VMware di Azure non necessitano di una connessione ExpressRoute se non è disponibile nell'ambiente di origine.  Se si decide di ospitare i carichi di lavoro in più aree di Azure, è possibile eseguire migrazioni in più posizioni.

Per ulteriori informazioni su queste soluzioni, vedere [RiverMeadow](https://www.rivermeadow.com/migrating-to-vmware-on-azure).

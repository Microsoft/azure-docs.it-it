# <a name="supported-browsers"></a>Browser supportati 

Azure competenza supporta i seguenti browser. È consigliabile usare il browser più aggiornato che sia compatibile con il sistema operativo. 

* Microsoft Edge (versione più recente)
* Safari (versione più recente, solo Mac)
* Chrome (versione più recente)
* Firefox (versione più recente)

## <a name="chrome-incognito-mode"></a>Modalità in incognito di Chrome

 I cookie di terze parti di blocco in incognito devono essere disabilitati per il funzionamento di ambito Studio.

## <a name="chromium-edge-inprivate-mode"></a>Modalità InPrivate cromo Edge

Per il funzionamento di questo strumento, è necessario disabilitare l'InPrivate di cromo Edge usando Strict Tracking Prevention.

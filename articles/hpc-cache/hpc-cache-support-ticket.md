# <a name="open-a-support-ticket-for-azure-hpc-cache"></a>Aprire un ticket di supporto per la cache HPC di AzureOpen a support ticket for Azure HPC Cache

Usare il portale di Azure per aprire un ticket di supporto. Passa all'istanza della cache, quindi fai clic sul collegamento **Nuova richiesta di supporto** visualizzato nella parte inferiore della barra laterale.

Per aprire un ticket quando non si dispone di una cache attiva, usare la pagina principale Guida - Supporto dal portale di Azure.To open a ticket when you do not have an active cache, use the main Help - support page from the Azure portal. Aprire il menu del portale dal controllo nella parte superiore sinistra dello schermo, quindi scorrere verso il basso e fare clic su **Guida e supporto**.

Scegliere **Nuova richiesta di supporto** e selezionare Tecnico per la guida specifica della cache HPC di Azure.Choose New support request and select **Technical** for help specific to Azure HPC Cache.

Selezionare l'abbonamento dall'elenco.

Per trovare il servizio cache HPC di Azure, fare clic sul pulsante **Tutti i servizi** e cercare HPC.

![Screenshot della richiesta di supporto - scheda Nozioni di base, parzialmente compilata come descritto](media/hpc-cache-support-request.png)

Compila il resto dei campi con le tue informazioni e preferenze, quindi invia il biglietto quando sei pronto.

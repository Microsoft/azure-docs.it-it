# <a name="migrate-to-azure-app-service"></a>Eseguire la migrazione al servizio app Azure

Usando il [servizio app Migration Assistant](https://azure.microsoft.com/services/app-service/migration-assistant/) è possibile eseguire la migrazione dell'app locale nel servizio app Azure. Il servizio app Migration Assistant è progettato per semplificare il passaggio al cloud tramite una soluzione gratuita, semplice e rapida per la migrazione di applicazioni da locale al cloud.

Con app Azure Migration Assistant del servizio è possibile:

- Analizza l'URL dell'app per valutare se è un candidato valido per la migrazione
- Scaricare il Migration Assistant per iniziare la migrazione.
- Usare lo strumento per eseguire i controlli di conformità e la valutazione generale delle impostazioni di configurazione dell'app
- Eseguire la migrazione dell'app o del sito al servizio app Azure tramite lo strumento.

[Vedere come eseguire la migrazione di app Web al servizio app Azure](https://www.youtube.com/watch?v=9LBUmkUhmXU).

Passaggio successivo:  [eseguire la migrazione di un'applicazione Web locale al servizio app Azure](https://docs.microsoft.com/learn/modules/migrate-app-service-migration-assistant/)
